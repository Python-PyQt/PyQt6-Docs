The :sip:ref:`~PyQt6.QtPrintSupport.QAbstractPrintDialog` class provides a base implementation for print dialogs used to configure printers.

This class implements getter and setter functions that are used to customize settings shown in print dialogs, but it is not used directly. Use :sip:ref:`~PyQt6.QtPrintSupport.QPrintDialog` to display a print dialog in your application.

.. seealso:: :sip:ref:`~PyQt6.QtPrintSupport.QPrintDialog`, :sip:ref:`~PyQt6.QtPrintSupport.QPrinter`.
