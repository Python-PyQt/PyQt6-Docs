Creates a :sip:ref:`~PyQt6.QtDBus.QDBusServiceWatcher` object. Note that until you set a connection with :sip:ref:`~PyQt6.QtDBus.QDBusServiceWatcher.setConnection`, this object will not emit any signals.

The *parent* parameter is passed to :sip:ref:`~PyQt6.QtCore.QObject` to set the parent of this object.
