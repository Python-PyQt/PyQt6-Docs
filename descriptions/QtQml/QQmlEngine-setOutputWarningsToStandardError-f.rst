Set whether warning messages will be output to stderr to *enabled*.

If *enabled* is true, any warning messages generated by QML will be output to stderr and emitted by the :sip:ref:`~PyQt6.QtQml.QQmlEngine.warnings` signal. If *enabled* is false, only the :sip:ref:`~PyQt6.QtQml.QQmlEngine.warnings` signal will be emitted. This allows applications to handle warning output themselves.

The default value is true.

.. seealso:: :sip:ref:`~PyQt6.QtQml.QQmlEngine.outputWarningsToStandardError`.
