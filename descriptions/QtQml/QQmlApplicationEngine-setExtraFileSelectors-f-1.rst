Sets the *extraFileSelectors* to be passed to the internal :sip:ref:`~PyQt6.QtQml.QQmlFileSelector` used for resolving URLs to local files. The *extraFileSelectors* are applied when the first QML file is loaded. Setting them afterwards has no effect.
