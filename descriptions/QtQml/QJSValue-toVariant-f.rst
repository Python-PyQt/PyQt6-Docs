Returns the `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_ value of this :sip:ref:`~PyQt6.QtQml.QJSValue`, if it can be converted to a `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_; otherwise returns an invalid `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_. The conversion is performed according to the following table:

+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Input Type                                                                   | Result                                                                                                                                                                                            |
+==============================================================================+===================================================================================================================================================================================================+
| Undefined                                                                    | An invalid `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_.                                                                                                                 |
+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Null                                                                         | A `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_ containing a null pointer (\ :sip:ref:`~PyQt6.QtCore.QMetaType.Type.Nullptr`).                                            |
+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Boolean                                                                      | A `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_ containing the value of the boolean.                                                                                      |
+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Number                                                                       | A `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_ containing the value of the number.                                                                                       |
+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| String                                                                       | A `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_ containing the value of the string.                                                                                       |
+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_ Object | The result is the `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_ value of the object (no conversion).                                                                      |
+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtCore.QObject` Object                                      | A `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_ containing a pointer to the :sip:ref:`~PyQt6.QtCore.QObject`.                                                             |
+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Date Object                                                                  | A `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_ containing the date value (\ :sip:ref:`~PyQt6.QtQml.QJSValue.toDateTime`).                                                |
+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| RegExp Object                                                                | A `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_ containing the regular expression value.                                                                                  |
+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Array Object                                                                 | The array is converted to a QVariantList. Each element is converted to a `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_, recursively; cyclic references are not followed.  |
+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Object                                                                       | The object is converted to a QVariantMap. Each property is converted to a `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_, recursively; cyclic references are not followed. |
+------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
