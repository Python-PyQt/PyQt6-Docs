Specifies which transparency technique to use. The Default value is ``Default``. When rendering transparent surface graphs, use ``Approximate`` or ``Accurate``.

**Note:** Accurate transparency is not yet implemented and will be enabled when the required functionality is added to `QtQuick3D <https://doc.qt.io/qt-6/qtquick3d-qmlmodule.html>`_.
