The :sip:ref:`~PyQt6.QtGraphs.QSurfaceDataItem` class provides a container for resolved data to be added to surface graphs.

A surface data item holds the data for a single vertex in a surface graph. Surface data proxies parse data into :sip:ref:`~PyQt6.QtGraphs.QSurfaceDataItem` instances for visualization.

.. seealso:: `QSurfaceDataProxy <https://doc.qt.io/qt-6/qsurfacedataproxy-qtgraphs-proxy.html>`_, :sip:ref:`~PyQt6.Qt Data Visualization C++ Classes`.
