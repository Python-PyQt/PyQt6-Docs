The :sip:ref:`~PyQt6.QtGraphs.QBarDataItem` class provides a container for resolved data to be added to bar graphs.

A bar data item holds the data for a single rendered bar in a graph. Bar data proxies parse data into :sip:ref:`~PyQt6.QtGraphs.QBarDataItem` instances for visualization.

.. seealso:: `QBarDataProxy <https://doc.qt.io/qt-6/qbardataproxy-qtgraphs-proxy.html>`_, :sip:ref:`~PyQt6.Qt Data Visualization C++ Classes`.
