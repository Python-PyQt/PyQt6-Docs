The :sip:ref:`~PyQt6.QtGraphs.QAbstract3DSeries` class is a base class for all 3D data series.

There are inherited classes for each supported series type: :sip:ref:`~PyQt6.QtGraphs.QBar3DSeries`, :sip:ref:`~PyQt6.QtGraphs.QScatter3DSeries`, and :sip:ref:`~PyQt6.QtGraphs.QSurface3DSeries`.

For more information, see `Qt Graphs Data Handling with 3D <https://doc.qt.io/qt-6/qtgraphs-data-handling.html>`_.
