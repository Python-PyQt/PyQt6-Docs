The :sip:ref:`~PyQt6.QtGraphs.QBarCategoryAxis` class adds categories to a graph's axes.

:sip:ref:`~PyQt6.QtGraphs.QBarCategoryAxis` can be set up to show an axis line with tick marks, grid lines, and shades. Categories are drawn between the ticks.
