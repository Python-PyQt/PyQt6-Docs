The :sip:ref:`~PyQt6.QtGraphs.QXYSeries` class is a parent class for all x & y series classes.

In :sip:ref:`~PyQt6.QtGraphs.QXYSeries`, data points are defined as a list of :sip:ref:`~PyQt6.QtCore.QPointF`, defining X and Y positions.

.. seealso:: :sip:ref:`~PyQt6.QtGraphs.QLineSeries`, :sip:ref:`~PyQt6.QtGraphs.QScatterSeries`.
