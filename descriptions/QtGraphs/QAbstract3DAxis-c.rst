The :sip:ref:`~PyQt6.QtGraphs.QAbstract3DAxis` class is a base class for the axes of a 3D graph.

This class specifies the enumerations, properties, and functions shared by graph axes. It should not be used directly, but one of its subclasses should be used instead.

.. seealso:: :sip:ref:`~PyQt6.QtGraphs.QCategory3DAxis`, :sip:ref:`~PyQt6.QtGraphs.QValue3DAxis`.
