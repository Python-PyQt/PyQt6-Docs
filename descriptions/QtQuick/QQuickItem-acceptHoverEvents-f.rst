Returns whether hover events are accepted by this item.

The default value is false.

If this is false, then the item will not receive any hover events through the :sip:ref:`~PyQt6.QtQuick.QQuickItem.hoverEnterEvent`, :sip:ref:`~PyQt6.QtQuick.QQuickItem.hoverMoveEvent` and :sip:ref:`~PyQt6.QtQuick.QQuickItem.hoverLeaveEvent` functions.
