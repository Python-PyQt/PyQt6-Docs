Sets the level of anisotropic filtering to *level*. The default value is :sip:ref:`~PyQt6.QtQuick.QSGTexture.AnisotropyLevel.AnisotropyNone`, which means no anisotropic filtering is enabled.

**Note:** The request may be ignored depending on the graphics API in use. There is no guarantee anisotropic filtering is supported at run time.
