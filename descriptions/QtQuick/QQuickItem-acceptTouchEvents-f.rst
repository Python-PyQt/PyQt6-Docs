Returns whether touch events are accepted by this item.

The default value is ``false``.

If this is ``false``, then the item will not receive any touch events through the :sip:ref:`~PyQt6.QtQuick.QQuickItem.touchEvent` function.

.. seealso:: :sip:ref:`~PyQt6.QtQuick.QQuickItem.setAcceptTouchEvents`.
