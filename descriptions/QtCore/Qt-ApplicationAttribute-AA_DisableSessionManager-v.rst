Disables the :sip:ref:`~PyQt6.QtGui.QSessionManager`. By default Qt will connect to a running session manager for a GUI application on supported platforms, use of a session manager may be redundant for system services. This attribute must be set before :sip:ref:`~PyQt6.QtGui.QGuiApplication` is constructed. This value was added in 5.14
