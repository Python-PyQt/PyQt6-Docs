Constructs a null time object. For a null time, :sip:ref:`~PyQt6.QtCore.QTime.isNull` returns ``true`` and :sip:ref:`~PyQt6.QtCore.QTime.isValid` returns ``false``. If you need a zero time, use :sip:ref:`~PyQt6.QtCore.QTime`\ (0, 0). For the start of a day, see :sip:ref:`~PyQt6.QtCore.QDate.startOfDay`.
