Adds _NET_WM_WINDOW_TYPE_DROPDOWN_MENU to the window's _NET_WM_WINDOW_TYPE X11 window property. See http://standards.freedesktop.org/wm-spec/ for more details. This attribute has no effect on non-X11 platforms. **Note**: Qt automatically sets this attribute for :sip:ref:`~PyQt6.QtWidgets.QMenu` objects added to a :sip:ref:`~PyQt6.QtWidgets.QMenuBar`.
