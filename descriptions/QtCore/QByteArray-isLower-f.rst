Returns ``true`` if this byte array is lowercase, that is, if it's identical to its :sip:ref:`~PyQt6.QtCore.QByteArray.toLower` folding.

Note that this does *not* mean that the byte array only contains lowercase letters; only that it contains no ASCII uppercase letters.

.. seealso:: :sip:ref:`~PyQt6.QtCore.QByteArray.isUpper`, :sip:ref:`~PyQt6.QtCore.QByteArray.toLower`.
