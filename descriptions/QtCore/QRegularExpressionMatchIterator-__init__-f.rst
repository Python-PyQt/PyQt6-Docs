Constructs an empty, valid :sip:ref:`~PyQt6.QtCore.QRegularExpressionMatchIterator` object. The regular expression is set to a default-constructed one; the match type to :sip:ref:`~PyQt6.QtCore.QRegularExpression.MatchType.NoMatch` and the match options to :sip:ref:`~PyQt6.QtCore.QRegularExpression.MatchOptions.NoMatchOption`.

Invoking the :sip:ref:`~PyQt6.QtCore.QRegularExpressionMatchIterator.hasNext` member function on the constructed object will return false, as the iterator is not iterating on a valid sequence of matches.
