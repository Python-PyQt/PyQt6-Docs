Sets the current write channel of the :sip:ref:`~PyQt6.QtCore.QIODevice` to the given *channel*. The current output channel is used by the functions :sip:ref:`~PyQt6.QtCore.QIODevice.write`, :sip:ref:`~PyQt6.QtCore.QIODevice.putChar`. It also determines which channel triggers :sip:ref:`~PyQt6.QtCore.QIODevice` to emit :sip:ref:`~PyQt6.QtCore.QIODevice.bytesWritten`.
