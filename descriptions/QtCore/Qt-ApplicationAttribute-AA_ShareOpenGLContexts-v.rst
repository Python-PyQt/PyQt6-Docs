Enables resource sharing between the OpenGL contexts used by classes like :sip:ref:`~PyQt6.QtOpenGLWidgets.QOpenGLWidget` and :sip:ref:`~PyQt6.QtQuickWidgets.QQuickWidget`. This allows sharing OpenGL resources, like textures, between :sip:ref:`~PyQt6.QtOpenGLWidgets.QOpenGLWidget` instances that belong to different top-level windows. This attribute must be set before :sip:ref:`~PyQt6.QtGui.QGuiApplication` is constructed. This value was added in Qt 5.4.
