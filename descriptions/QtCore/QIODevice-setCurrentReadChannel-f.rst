Sets the current read channel of the :sip:ref:`~PyQt6.QtCore.QIODevice` to the given *channel*. The current input channel is used by the functions :sip:ref:`~PyQt6.QtCore.QIODevice.read`, :sip:ref:`~PyQt6.QtCore.QIODevice.readAll`, :sip:ref:`~PyQt6.QtCore.QIODevice.readLine`, and :sip:ref:`~PyQt6.QtCore.QIODevice.getChar`. It also determines which channel triggers :sip:ref:`~PyQt6.QtCore.QIODevice` to emit :sip:ref:`~PyQt6.QtCore.QIODevice.readyRead`.
