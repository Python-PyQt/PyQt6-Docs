Makes Qt quit the application when the last widget with the attribute set has accepted closeEvent(). This behavior can be modified with the :sip:ref:`~PyQt6.QtGui.QGuiApplication.quitOnLastWindowClosed` property. By default this attribute is set for all widgets of type :sip:ref:`~PyQt6.QtCore.Qt.WindowFlags.Window`.
