`QProcess <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qprocess>`_ merges the output of the running process into the standard output channel (``stdout``). The standard error channel (``stderr``) will not receive any data. The standard output and standard error data of the running process are interleaved.
