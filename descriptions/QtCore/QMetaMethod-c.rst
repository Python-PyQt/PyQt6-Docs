The :sip:ref:`~PyQt6.QtCore.QMetaMethod` class provides meta-data about a member function.

A :sip:ref:`~PyQt6.QtCore.QMetaMethod` has a :sip:ref:`~PyQt6.QtCore.QMetaMethod.methodType`, a :sip:ref:`~PyQt6.QtCore.QMetaMethod.methodSignature`, a list of :sip:ref:`~PyQt6.QtCore.QMetaMethod.parameterTypes` and :sip:ref:`~PyQt6.QtCore.QMetaMethod.parameterNames`, a return :sip:ref:`~PyQt6.QtCore.QMetaMethod.typeName`, a :sip:ref:`~PyQt6.QtCore.QMetaMethod.tag`, and an :sip:ref:`~PyQt6.QtCore.QMetaMethod.access` specifier. You can use invoke() to invoke the method on an arbitrary :sip:ref:`~PyQt6.QtCore.QObject`.

.. seealso:: :sip:ref:`~PyQt6.QtCore.QMetaObject`, :sip:ref:`~PyQt6.QtCore.QMetaEnum`, :sip:ref:`~PyQt6.QtCore.QMetaProperty`, `Qt's Property System <https://doc.qt.io/qt-6/properties.html>`_.
