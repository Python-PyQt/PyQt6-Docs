The user input resolution options define how :sip:ref:`~PyQt6.QtCore.QUrl.fromUserInput` should interpret strings that could either be a relative path or the short form of a HTTP URL. For instance ``file.pl`` can be either a local file or the URL ``http://file.pl``.
