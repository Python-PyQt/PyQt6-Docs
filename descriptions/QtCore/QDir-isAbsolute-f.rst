Returns ``true`` if the directory's path is absolute; otherwise returns ``false``. See :sip:ref:`~PyQt6.QtCore.QDir.isAbsolutePath`.

**Note:** Paths starting with a colon (\ *:*) are always considered absolute, as they denote a :sip:ref:`~PyQt6.QtCore.QResource`.
