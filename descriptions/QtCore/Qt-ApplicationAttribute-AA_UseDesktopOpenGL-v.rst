Forces the usage of desktop OpenGL (for example, *opengl32.dll* or *libGL.so*) on platforms that use dynamic loading of the OpenGL implementation. This attribute must be set before :sip:ref:`~PyQt6.QtGui.QGuiApplication` is constructed. This value was added in Qt 5.3.
