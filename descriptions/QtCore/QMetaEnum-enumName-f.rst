Returns the enum name of the flag (without the scope).

For example, the :sip:ref:`~PyQt6.QtCore.Qt.Alignment` flag has ``AlignmentFlag`` as the enum name, but ``Alignment`` as as the type name. Non flag enums has the same type and enum names.

Enum names have the same scope as the type name.
