The :sip:ref:`~PyQt6.QtCore.QDynamicPropertyChangeEvent` class contains event parameters for dynamic property change events.

Dynamic property change events are sent to objects when properties are dynamically added, changed or removed using :sip:ref:`~PyQt6.QtCore.QObject.setProperty`.
