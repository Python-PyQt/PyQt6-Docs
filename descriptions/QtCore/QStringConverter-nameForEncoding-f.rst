Returns the canonical name for encoding *e* or ``nullptr`` if *e* is an invalid value.

**Note:** In Qt versions prior to 6.10, 6.9.1, 6.8.4 or 6.5.9, calling this function with an invalid argument resulted in undefined behavior. Since the above-mentioned Qt versions, it returns nullptr instead.
