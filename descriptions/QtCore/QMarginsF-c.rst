The :sip:ref:`~PyQt6.QtCore.QMarginsF` class defines the four margins of a rectangle.

:sip:ref:`~PyQt6.QtCore.QMarginsF` defines a set of four margins; left, top, right and bottom, that describe the size of the borders surrounding a rectangle.

The :sip:ref:`~PyQt6.QtCore.QMarginsF.isNull` function returns ``true`` only if all margins are very close to zero.

:sip:ref:`~PyQt6.QtCore.QMarginsF` objects can be streamed as well as compared.
