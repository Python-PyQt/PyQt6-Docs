Returns ``true`` if this byte array is uppercase, that is, if it's identical to its :sip:ref:`~PyQt6.QtCore.QByteArray.toUpper` folding.

Note that this does *not* mean that the byte array only contains uppercase letters; only that it contains no ASCII lowercase letters.

.. seealso:: :sip:ref:`~PyQt6.QtCore.QByteArray.isLower`, :sip:ref:`~PyQt6.QtCore.QByteArray.toUpper`.
