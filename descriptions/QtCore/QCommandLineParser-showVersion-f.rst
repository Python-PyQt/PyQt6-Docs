Displays the version information from :sip:ref:`~PyQt6.QtCore.QCoreApplication.applicationVersion`, and exits the application. This is automatically triggered by the --version option, but can also be used to display the version when not using :sip:ref:`~PyQt6.QtCore.QCommandLineParser.process`. The exit code is set to EXIT_SUCCESS (0).
