By default, Qt Style Sheets disable regular `QWidget <https://doc.qt.io/qt-6/widgets-changes-qt6.html#qwidget>`_ palette and font propagation. When this flag is enabled, font and palette changes propagate as though the user had manually called the corresponding `QWidget <https://doc.qt.io/qt-6/widgets-changes-qt6.html#qwidget>`_ methods. See `The Style Sheet Syntax - Inheritance <https://doc.qt.io/qt-6/stylesheet-syntax.html#inheritance>`_ for more details. This value was added in Qt 5.7.
