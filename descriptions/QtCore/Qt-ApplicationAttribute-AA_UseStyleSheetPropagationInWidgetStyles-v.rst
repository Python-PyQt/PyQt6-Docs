By default, Qt Style Sheets disable regular :sip:ref:`~PyQt6.QtWidgets.QWidget` palette and font propagation. When this flag is enabled, font and palette changes done from a style sheet will propagate a single time, when the style sheet is set. See `The Style Sheet Syntax - Inheritance <https://doc.qt.io/qt-6/stylesheet-syntax.html#inheritance>`_ for more details. This value was added in Qt 5.7.
