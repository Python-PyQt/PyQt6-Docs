The :sip:ref:`~PyQt6.QtCore.QCryptographicHash` class provides a way to generate cryptographic hashes.

:sip:ref:`~PyQt6.QtCore.QCryptographicHash` can be used to generate cryptographic hashes of binary or text data.

Refer to the documentation of the :sip:ref:`~PyQt6.QtCore.QCryptographicHash.Algorithm` enum for a list of the supported algorithms.
