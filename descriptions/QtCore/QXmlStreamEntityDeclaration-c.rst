The :sip:ref:`~PyQt6.QtCore.QXmlStreamEntityDeclaration` class represents a DTD entity declaration.

An entity declaration consists of a :sip:ref:`~PyQt6.QtCore.QXmlStreamEntityDeclaration.name`, a :sip:ref:`~PyQt6.QtCore.QXmlStreamEntityDeclaration.notationName`, a :sip:ref:`~PyQt6.QtCore.QXmlStreamEntityDeclaration.systemId`, a :sip:ref:`~PyQt6.QtCore.QXmlStreamEntityDeclaration.publicId`, and a :sip:ref:`~PyQt6.QtCore.QXmlStreamEntityDeclaration.value`.
