Converts the value to a :sip:ref:`~PyQt6.QtCore.QVariant.__init__`.

The :sip:ref:`~PyQt6.QtCore.QJsonValue` types will be converted as follows:

+-----------+----------------------------------------------------------------------------------------------------+
| Constant  | Description                                                                                        |
+===========+====================================================================================================+
| Null      | :sip:ref:`~PyQt6.QtCore.QMetaType.Type.Nullptr`                                                    |
+-----------+----------------------------------------------------------------------------------------------------+
| Bool      | :sip:ref:`~PyQt6.QtCore.QMetaType.Type.Bool`                                                       |
+-----------+----------------------------------------------------------------------------------------------------+
| Double    | :sip:ref:`~PyQt6.QtCore.QMetaType.Type.Double` or :sip:ref:`~PyQt6.QtCore.QMetaType.Type.LongLong` |
+-----------+----------------------------------------------------------------------------------------------------+
| String    | QString                                                                                            |
+-----------+----------------------------------------------------------------------------------------------------+
| Array     | QVariantList                                                                                       |
+-----------+----------------------------------------------------------------------------------------------------+
| Object    | QVariantMap                                                                                        |
+-----------+----------------------------------------------------------------------------------------------------+
| Undefined | :sip:ref:`~PyQt6.QtCore.QVariant.__init__`                                                         |
+-----------+----------------------------------------------------------------------------------------------------+

.. seealso:: :sip:ref:`~PyQt6.QtCore.QJsonValue.fromVariant`.
