Returns the name of the type stored in the variant. The returned strings describe the C++ datatype used to store the data: for example, "`QFont <https://doc.qt.io/qt-6/gui-changes-qt6.html#qfont>`_", "QString", or "QVariantList". An Invalid variant returns 0.
