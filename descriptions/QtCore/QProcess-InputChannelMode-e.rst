This enum describes the process input channel modes of `QProcess <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qprocess>`_. Pass one of these values to :sip:ref:`~PyQt6.QtCore.QProcess.setInputChannelMode` to set the current write channel mode.
