`QProcess <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qprocess>`_ manages the output of the running process, keeping standard output and standard error data in separate internal buffers. You can select the `QProcess <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qprocess>`_'s current read channel by calling :sip:ref:`~PyQt6.QtCore.QProcess.setReadChannel`. This is the default channel mode of `QProcess <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qprocess>`_.
