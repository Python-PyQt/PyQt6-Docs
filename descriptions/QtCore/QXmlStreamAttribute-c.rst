The :sip:ref:`~PyQt6.QtCore.QXmlStreamAttribute` class represents a single XML attribute.

An attribute consists of an optionally empty :sip:ref:`~PyQt6.QtCore.QXmlStreamAttribute.namespaceUri`, a :sip:ref:`~PyQt6.QtCore.QXmlStreamAttribute.name`, a :sip:ref:`~PyQt6.QtCore.QXmlStreamAttribute.value`, and an :sip:ref:`~PyQt6.QtCore.QXmlStreamAttribute.isDefault` attribute.

The raw XML attribute name is returned as :sip:ref:`~PyQt6.QtCore.QXmlStreamAttribute.qualifiedName`.
