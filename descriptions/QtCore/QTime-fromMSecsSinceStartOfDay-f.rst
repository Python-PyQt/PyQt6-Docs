Returns a new :sip:ref:`~PyQt6.QtCore.QTime` instance with the time set to the number of *msecs* since the start of the day, i.e. since 00:00:00.

If *msecs* falls outside the valid range an invalid :sip:ref:`~PyQt6.QtCore.QTime` will be returned.
