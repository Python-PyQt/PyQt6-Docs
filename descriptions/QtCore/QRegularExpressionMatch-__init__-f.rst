Constructs a valid, empty :sip:ref:`~PyQt6.QtCore.QRegularExpressionMatch` object. The regular expression is set to a default-constructed one; the match type to :sip:ref:`~PyQt6.QtCore.QRegularExpression.MatchType.NoMatch` and the match options to :sip:ref:`~PyQt6.QtCore.QRegularExpression.MatchOptions.NoMatchOption`.

The object will report no match through the :sip:ref:`~PyQt6.QtCore.QRegularExpressionMatch.hasMatch` and the :sip:ref:`~PyQt6.QtCore.QRegularExpressionMatch.hasPartialMatch` member functions.
