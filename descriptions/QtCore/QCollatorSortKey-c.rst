The :sip:ref:`~PyQt6.QtCore.QCollatorSortKey` class can be used to speed up string collation.

The :sip:ref:`~PyQt6.QtCore.QCollatorSortKey` class is always created by :sip:ref:`~PyQt6.QtCore.QCollator.sortKey` and is used for fast strings collation, for example when collating many strings.
