Sets the current read channel of the `QProcess <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qprocess>`_ to the given *channel*. The current input channel is used by the functions read(), readAll(), readLine(), and getChar(). It also determines which channel triggers `QProcess <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qprocess>`_ to emit readyRead().
