Adds _NET_WM_WINDOW_TYPE_COMBO to the window's _NET_WM_WINDOW_TYPE X11 window property. See http://standards.freedesktop.org/wm-spec/ for more details. This attribute has no effect on non-X11 platforms. **Note**: Qt automatically sets this attribute for the `QComboBox <https://doc.qt.io/qt-6/widgets-changes-qt6.html#qcombobox>`_ pop-up.
