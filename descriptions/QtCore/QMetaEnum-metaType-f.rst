Returns the meta type of the enum.

If the :sip:ref:`~PyQt6.QtCore.QMetaObject` this enum is part of was generated with Qt 6.5 or earlier this will be the invalid metatype.

**Note:** This is the meta type of the enum itself, not of its underlying numeric type. You can retrieve the meta type of the underlying type of the enum using :sip:ref:`~PyQt6.QtCore.QMetaType.underlyingType`.
