Adds _NET_WM_WINDOW_TYPE_DIALOG to the window's _NET_WM_WINDOW_TYPE X11 window property. See http://standards.freedesktop.org/wm-spec/ for more details. This attribute has no effect on non-X11 platforms. **Note**: Qt automatically sets this attribute for the :sip:ref:`~PyQt6.QtCore.Qt.WindowFlags.Dialog` and :sip:ref:`~PyQt6.QtCore.Qt.WindowFlags.Sheet` window types.
