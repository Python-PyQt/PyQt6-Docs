This enum defines a constant that can be given as precision to QString::number(), :sip:ref:`~PyQt6.QtCore.QByteArray.number`, and :sip:ref:`~PyQt6.QtCore.QLocale.toString` when converting floats or doubles, in order to express a variable number of digits as precision.
