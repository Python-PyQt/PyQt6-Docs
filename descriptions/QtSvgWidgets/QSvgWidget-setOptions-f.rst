Sets the widget's renderer options to *options*.

This property holds a set of :sip:ref:`~PyQt6.QtSvg.QtSvg.Option` flags that can be used to enable or disable various features of the parsing and rendering of SVG files. It must be set before calling the load function to have any effect.

By default, no flags are set.

.. seealso:: :sip:ref:`~PyQt6.QtSvgWidgets.QSvgWidget.options`.
