The :sip:ref:`~PyQt6.QtNetworkAuth.QAbstractOAuth2` class is the base of all implementations of OAuth 2 authentication methods.

The class defines the basic interface of the OAuth 2 authentication classes. By inheriting this class, you can create custom authentication methods using the OAuth 2 standard for different web services.

A description of how OAuth 2 works can be found in: `The OAuth 2.0 Authorization Framework <https://tools.ietf.org/html/rfc6749>`_
