.. sip:module-description::
    :status:    done
    :brief:     Classes for OAuth-based authorization to online services

The :sip:ref:`~PyQt6.QtNetworkAuth` module contains classes that enable PyQt
applications to obtain limited access to online accounts and HTTP services
without exposing users' passwords.

Currently, the supported authorization protocol is `OAuth
<https://oauth.net>`__ versions 1 and 2.
