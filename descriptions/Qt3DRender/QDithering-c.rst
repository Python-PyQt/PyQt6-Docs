Enable dithering.

A :sip:ref:`~PyQt6.Qt3DRender.QDithering` class enables dithering. Dithering adds noise to the color values to randomize quantization error in order to prevent large scale patterns in the final image, such as banding. Dithering is most useful when rendering to a surface with low color bit depth, such as RGB565 or RGBA4444.
