The :sip:ref:`~PyQt6.Qt3DRender.QRenderTargetOutput` class allows the specification of an attachment of a render target (whether it is a color texture, a depth texture, etc... ).

A :sip:ref:`~PyQt6.Qt3DRender.QRenderTargetOutput` specifies the attachment point and parameters for texture that is attached to render target. In addition to the attachment point, texture miplevel, layer and cubemap face can be specified. The texture attached to the :sip:ref:`~PyQt6.Qt3DRender.QRenderTargetOutput` must be compatible with the given parameters.
