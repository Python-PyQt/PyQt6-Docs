The :sip:ref:`~PyQt6.Qt3DRender.QStencilTest` class specifies arguments for the stecil test.

A :sip:ref:`~PyQt6.Qt3DRender.QStencilTest` class specifies argument for the stencil test. The stencil test comprises of three arguments: stencil test function, stencil test mask and stencil reference value. :sip:ref:`~PyQt6.Qt3DRender.QStencilTest` allows these arguments to be set for both front- and back-facing polygons separately.
