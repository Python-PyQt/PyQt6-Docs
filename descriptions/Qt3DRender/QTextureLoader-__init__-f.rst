Constructs a new :sip:ref:`~PyQt6.Qt3DRender.QTextureLoader` instance with *parent* as parent.

Note that by default, if not contradicted by the file metadata, the loaded texture will have the following properties set: - wrapMode set to Repeat - minificationFilter set to :sip:ref:`~PyQt6.Qt3DRender.QAbstractTexture.Filter.LinearMipMapLinear` - magnificationFilter set to Linear - generateMipMaps set to true - maximumAnisotropy set to 16.0f - target set to :sip:ref:`~PyQt6.Qt3DRender.QAbstractTexture.Target.TargetAutomatic`
