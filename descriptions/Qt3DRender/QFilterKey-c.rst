The :sip:ref:`~PyQt6.Qt3DRender.QFilterKey` class provides storage for filter keys and their values.

Filter keys are used by QTechnique and QRenderPass to specify at which stage of rendering the technique or the render pass is used.

**Note:** :sip:ref:`~PyQt6.Qt3DRender.QFilterKey` node can not be disabled.
