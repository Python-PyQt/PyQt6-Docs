The :sip:ref:`~PyQt6.Qt3DRender.QRenderSettings` class holds settings related to rendering process and host the active `FrameGraph <https://doc.qt.io/qt-6/qt3drender-framegraph.html>`_.

The :sip:ref:`~PyQt6.Qt3DRender.QRenderSettings` component must be set as a component of the scene root entity. It specifies render policy and picking settings, as well as hosts the active `FrameGraph <https://doc.qt.io/qt-6/qt3drender-framegraph.html>`_.
