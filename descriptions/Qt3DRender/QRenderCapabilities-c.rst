The :sip:ref:`~PyQt6.Qt3DRender.QRenderCapabilities` class holds settings related to available rendering engines.

:sip:ref:`~PyQt6.Qt3DRender.QRenderCapabilities` provides details of graphical features that are available at runtime. It can be used to decide which code path to use for some algorithms, for example, depending on whether compute shaders are available or not.
