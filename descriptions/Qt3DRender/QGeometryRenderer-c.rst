Encapsulates geometry rendering.

A :sip:ref:`~PyQt6.Qt3DRender.QGeometryRenderer` holds all the information necessary to draw a :sip:ref:`~PyQt6.Qt3DCore.QGeometry`. A QGeometry holds the coordinates of the geometry data - :sip:ref:`~PyQt6.Qt3DRender.QGeometryRenderer` specifies how to interpret that data.
