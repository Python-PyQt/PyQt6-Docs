The :sip:ref:`~PyQt6.QtCharts.QXYLegendMarker` class is a legend marker for a line, spline, or scatter series.

An XY legend marker is related to :sip:ref:`~PyQt6.QtCharts.QXYSeries` derived classes: :sip:ref:`~PyQt6.QtCharts.QLineSeries`, :sip:ref:`~PyQt6.QtCharts.QSplineSeries`, and :sip:ref:`~PyQt6.QtCharts.QScatterSeries`. Each marker is related to one series.

.. seealso:: :sip:ref:`~PyQt6.QtCharts.QLegend`, :sip:ref:`~PyQt6.QtCharts.QXYSeries`, :sip:ref:`~PyQt6.QtCharts.QSplineSeries`, :sip:ref:`~PyQt6.QtCharts.QScatterSeries`, :sip:ref:`~PyQt6.QtCharts.QLineSeries`.
