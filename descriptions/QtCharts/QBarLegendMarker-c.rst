The :sip:ref:`~PyQt6.QtCharts.QBarLegendMarker` class is a legend marker for a bar series.

A bar legend marker is related to :sip:ref:`~PyQt6.QtCharts.QAbstractBarSeries` derived classes. With a bar series, each marker is related to one :sip:ref:`~PyQt6.QtCharts.QBarSet`.

.. seealso:: :sip:ref:`~PyQt6.QtCharts.QLegend`, :sip:ref:`~PyQt6.QtCharts.QAbstractBarSeries`, :sip:ref:`~PyQt6.QtCharts.QBarSet`.
