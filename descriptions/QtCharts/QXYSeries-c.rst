The :sip:ref:`~PyQt6.QtCharts.QXYSeries` class is a base class for line, spline, and scatter series.

:sip:ref:`~PyQt6.QtCharts.QXYSeries` supports displaying best fit line on a chart. Best fit line is a line through a chart that expresses the relationship between points.
