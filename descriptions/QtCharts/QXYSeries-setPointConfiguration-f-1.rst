Enables customizing a particular aspect of a point's configuration.

**Note:** Points configuration concept provides a flexible way to configure various aspects of a point's appearance. Thus, values need to have an elastic type such as :sip:ref:`~PyQt6.QtCore.QVariant`. See :sip:ref:`~PyQt6.QtCharts.QXYSeries.PointConfiguration` to see what *value* should be passed for certain *key*.
