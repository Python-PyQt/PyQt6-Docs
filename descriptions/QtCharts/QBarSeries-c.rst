The :sip:ref:`~PyQt6.QtCharts.QBarSeries` class presents a series of data as vertical bars grouped by category.

This class draws data as a series of vertical bars grouped by category, with one bar per category from each bar set added to the series.

See the `bar chart example <https://doc.qt.io/qt-6/qtcharts-barchart-example.html>`_ to learn how to create a grouped bar chart.
