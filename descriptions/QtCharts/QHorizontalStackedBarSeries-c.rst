The :sip:ref:`~PyQt6.QtCharts.QHorizontalStackedBarSeries` class presents a series of data as horizontally stacked bars, with one bar per category.

Each bar set added to the series contributes a single segment to each stacked bar.

See the `Charts with Widgets Gallery <https://doc.qt.io/qt-6/qtcharts-chartsgallery-example.html>`_ to learn how to create a horizontal stacked bar chart.
