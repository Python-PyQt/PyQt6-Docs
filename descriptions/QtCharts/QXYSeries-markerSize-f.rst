The default size depends on the specific :sip:ref:`~PyQt6.QtCharts.QXYSeries` type. :sip:ref:`~PyQt6.QtCharts.QScatterSeries` has a default of 15.0 :sip:ref:`~PyQt6.QtCharts.QLineSeries` has a default of the series pen size \* 1.5
