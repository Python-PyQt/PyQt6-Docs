The :sip:ref:`~PyQt6.QtCharts.QStackedBarSeries` class presents a series of data as vertically stacked bars, with one bar per category.

Each bar set added to the series contributes a single segment to each stacked bar.

See the `stacked bar chart example <https://doc.qt.io/qt-6/qtcharts-stackedbarchart-example.html>`_ to learn how to create a stacked bar chart.
