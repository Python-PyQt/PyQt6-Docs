Sets axis labels editability to *editable*.

When the labels are editable the user will be able to change the range of the axis conveniently by editing any of the labels. This feature is only supported for the :sip:ref:`~PyQt6.QtCharts.QValueAxis` and the :sip:ref:`~PyQt6.QtCharts.QDateTimeAxis`.

By default, labels are not editable.

.. seealso:: :sip:ref:`~PyQt6.QtCharts.QAbstractAxis.labelsEditable`.
