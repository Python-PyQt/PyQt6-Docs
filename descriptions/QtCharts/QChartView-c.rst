The :sip:ref:`~PyQt6.QtCharts.QChartView` is a standalone widget that can display charts.

A chart view does not require a :sip:ref:`~PyQt6.QtWidgets.QGraphicsScene` object to work. To display a chart in an existing :sip:ref:`~PyQt6.QtWidgets.QGraphicsScene`, the :sip:ref:`~PyQt6.QtCharts.QChart` or :sip:ref:`~PyQt6.QtCharts.QPolarChart` class should be used instead.

.. seealso:: :sip:ref:`~PyQt6.QtCharts.QChart`, :sip:ref:`~PyQt6.QtCharts.QPolarChart`.
