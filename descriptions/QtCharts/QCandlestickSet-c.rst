The :sip:ref:`~PyQt6.QtCharts.QCandlestickSet` class represents a single candlestick item in a candlestick chart.

Five values are needed to create a graphical representation of a candlestick item: *open*, *high*, *low*, *close*, and *timestamp*. These values can be either passed to a :sip:ref:`~PyQt6.QtCharts.QCandlestickSet` constructor or set by using :sip:ref:`~PyQt6.QtCharts.QCandlestickSet.setOpen`, :sip:ref:`~PyQt6.QtCharts.QCandlestickSet.setHigh`, :sip:ref:`~PyQt6.QtCharts.QCandlestickSet.setLow`, :sip:ref:`~PyQt6.QtCharts.QCandlestickSet.setClose`, and :sip:ref:`~PyQt6.QtCharts.QCandlestickSet.setTimestamp`.

.. seealso:: :sip:ref:`~PyQt6.QtCharts.QCandlestickSeries`.
