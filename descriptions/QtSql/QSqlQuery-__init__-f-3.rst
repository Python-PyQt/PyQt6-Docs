Constructs a `QSqlQuery <https://doc.qt.io/qt-6/qtsql-changes-qt6.html#qsqlquery>`_ object using the SQL *query* and the database *db*. If *db* is not specified, or is invalid, the application's default database is used. If *query* is not an empty string, it will be executed.
