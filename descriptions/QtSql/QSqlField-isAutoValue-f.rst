Returns ``true`` if the value is auto-generated by the database, for example auto-increment primary key values.

**Note:** When using the ODBC driver, due to limitations in the ODBC API, the ``isAutoValue()`` field is only populated in a :sip:ref:`~PyQt6.QtSql.QSqlField` resulting from a :sip:ref:`~PyQt6.QtSql.QSqlRecord` obtained by executing a ``SELECT`` query. It is ``false`` in a :sip:ref:`~PyQt6.QtSql.QSqlField` resulting from a :sip:ref:`~PyQt6.QtSql.QSqlRecord` returned from :sip:ref:`~PyQt6.QtSql.QSqlDatabase.record` or :sip:ref:`~PyQt6.QtSql.QSqlDatabase.primaryIndex`.
