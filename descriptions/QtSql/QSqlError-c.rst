The :sip:ref:`~PyQt6.QtSql.QSqlError` class provides SQL database error information.

A :sip:ref:`~PyQt6.QtSql.QSqlError` object can provide database-specific error data, including the :sip:ref:`~PyQt6.QtSql.QSqlError.driverText` and :sip:ref:`~PyQt6.QtSql.QSqlError.databaseText` messages (or both concatenated together as :sip:ref:`~PyQt6.QtSql.QSqlError.text`), and the :sip:ref:`~PyQt6.QtSql.QSqlError.nativeErrorCode` and :sip:ref:`~PyQt6.QtSql.QSqlError.type`.
