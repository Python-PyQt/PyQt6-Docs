This enum contains a list of features a driver might support. Use :sip:ref:`~PyQt6.QtSql.QSqlDriver.hasFeature` to query whether a feature is supported or not. Some features depend on the database server so they can only properly determined after the database connection is successfully opened with :sip:ref:`~PyQt6.QtSql.QSqlDatabase.open`.

More information about supported features can be found in the `Qt SQL driver <https://doc.qt.io/qt-6/sql-driver.html>`_ documentation.

.. seealso:: :sip:ref:`~PyQt6.QtSql.QSqlDriver.hasFeature`.
