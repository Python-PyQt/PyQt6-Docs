Returns the number of sample points that have been requested with :sip:ref:`~PyQt6.QtOpenGL.QOpenGLTimeMonitor.setSampleCount`. If create was successfully called following :sip:ref:`~PyQt6.QtOpenGL.QOpenGLTimeMonitor.setSampleCount`, then the value returned will be the actual number of sample points that can be used.

The default value for sample count is 2, leading to the measurement of a single interval.

.. seealso:: :sip:ref:`~PyQt6.QtOpenGL.QOpenGLTimeMonitor.setSampleCount`.
