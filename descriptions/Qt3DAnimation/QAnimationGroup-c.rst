A class grouping animations together.

:sip:ref:`~PyQt6.Qt3DAnimation.QAnimationGroup` class is used to group multiple animations so that they can act as one animation. The position set to the group is also set to all animations in a group. The duration is the maximum of the individual animations. The animations can be any supported animation type and do not have to have the same name.
