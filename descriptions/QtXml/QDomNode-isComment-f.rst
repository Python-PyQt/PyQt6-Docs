Returns ``true`` if the node is a comment; otherwise returns ``false``.

If this function returns ``true``, it does not imply that this object is a :sip:ref:`~PyQt6.QtXml.QDomComment`; you can get the :sip:ref:`~PyQt6.QtXml.QDomComment` with :sip:ref:`~PyQt6.QtXml.QDomNode.toComment`.
