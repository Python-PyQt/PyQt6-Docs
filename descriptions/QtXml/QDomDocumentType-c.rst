The :sip:ref:`~PyQt6.QtXml.QDomDocumentType` class is the representation of the DTD in the document tree.

The :sip:ref:`~PyQt6.QtXml.QDomDocumentType` class allows read-only access to some of the data structures in the DTD: it can return a map of all :sip:ref:`~PyQt6.QtXml.QDomDocumentType.entities` and :sip:ref:`~PyQt6.QtXml.QDomDocumentType.notations`. In addition the function :sip:ref:`~PyQt6.QtXml.QDomDocumentType.name` returns the name of the document type as specified in the &lt;!DOCTYPE name&gt; tag. This class also provides the :sip:ref:`~PyQt6.QtXml.QDomDocumentType.publicId`, :sip:ref:`~PyQt6.QtXml.QDomDocumentType.systemId` and :sip:ref:`~PyQt6.QtXml.QDomDocumentType.internalSubset` functions.

.. seealso:: :sip:ref:`~PyQt6.QtXml.QDomDocument`.
