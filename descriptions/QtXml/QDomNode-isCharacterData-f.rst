Returns ``true`` if the node is a character data node; otherwise returns ``false``.

If this function returns ``true``, it does not imply that this object is a :sip:ref:`~PyQt6.QtXml.QDomCharacterData`; you can get the :sip:ref:`~PyQt6.QtXml.QDomCharacterData` with :sip:ref:`~PyQt6.QtXml.QDomNode.toCharacterData`.
