Returns ``true`` if the node is a processing instruction; otherwise returns ``false``.

If this function returns ``true``, it does not imply that this object is a :sip:ref:`~PyQt6.QtXml.QDomProcessingInstruction`; you can get the QProcessingInstruction with :sip:ref:`~PyQt6.QtXml.QDomNode.toProcessingInstruction`.
