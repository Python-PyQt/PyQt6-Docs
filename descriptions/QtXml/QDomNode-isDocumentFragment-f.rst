Returns ``true`` if the node is a document fragment; otherwise returns false.

If this function returns ``true``, it does not imply that this object is a :sip:ref:`~PyQt6.QtXml.QDomDocumentFragment`; you can get the :sip:ref:`~PyQt6.QtXml.QDomDocumentFragment` with :sip:ref:`~PyQt6.QtXml.QDomNode.toDocumentFragment`.
