Returns ``true`` if the node is a CDATA section; otherwise returns false.

If this function returns ``true``, it does not imply that this object is a :sip:ref:`~PyQt6.QtXml.QDomCDATASection`; you can get the :sip:ref:`~PyQt6.QtXml.QDomCDATASection` with :sip:ref:`~PyQt6.QtXml.QDomNode.toCDATASection`.
