Imports the node *importedNode* from another document to this document. *importedNode* remains in the original document; this function creates a copy that can be used within this document.

This function returns the imported node that belongs to this document. The returned node has no parent. It is not possible to import :sip:ref:`~PyQt6.QtXml.QDomDocument` and :sip:ref:`~PyQt6.QtXml.QDomDocumentType` nodes. In those cases this function returns a :sip:ref:`~PyQt6.QtXml.QDomNode.isNull`.

If *importedNode* is a :sip:ref:`~PyQt6.QtXml.QDomNode.isNull`, a null node is returned.

If *deep* is true, this function imports not only the node *importedNode* but its whole subtree; if it is false, only the *importedNode* is imported. The argument *deep* has no effect on :sip:ref:`~PyQt6.QtXml.QDomAttr` and :sip:ref:`~PyQt6.QtXml.QDomEntityReference` nodes, since the descendants of :sip:ref:`~PyQt6.QtXml.QDomAttr` nodes are always imported and those of :sip:ref:`~PyQt6.QtXml.QDomEntityReference` nodes are never imported.

The behavior of this function is slightly different depending on the node types:

+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Node Type                                         | Behavior                                                                                                                                                                                                                                                                                |
+===================================================+=========================================================================================================================================================================================================================================================================================+
| :sip:ref:`~PyQt6.QtXml.QDomAttr`                  | The owner element is set to 0 and the specified flag is set to true in the generated attribute. The whole subtree of *importedNode* is always imported for attribute nodes: *deep* has no effect.                                                                                       |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtXml.QDomDocument`              | Document nodes cannot be imported.                                                                                                                                                                                                                                                      |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtXml.QDomDocumentFragment`      | If *deep* is true, this function imports the whole document fragment; otherwise it only generates an empty document fragment.                                                                                                                                                           |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtXml.QDomDocumentType`          | Document type nodes cannot be imported.                                                                                                                                                                                                                                                 |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtXml.QDomElement`               | Attributes for which :sip:ref:`~PyQt6.QtXml.QDomAttr.specified` is true are also imported, other attributes are not imported. If *deep* is true, this function also imports the subtree of *importedNode*; otherwise it imports only the element node (and some attributes, see above). |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtXml.QDomEntity`                | Entity nodes can be imported, but at the moment there is no way to use them since the document type is read-only in DOM level 2.                                                                                                                                                        |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtXml.QDomEntityReference`       | Descendants of entity reference nodes are never imported: *deep* has no effect.                                                                                                                                                                                                         |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtXml.QDomNotation`              | Notation nodes can be imported, but at the moment there is no way to use them since the document type is read-only in DOM level 2.                                                                                                                                                      |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtXml.QDomProcessingInstruction` | The target and value of the processing instruction is copied to the new node.                                                                                                                                                                                                           |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtXml.QDomText`                  | The text is copied to the new node.                                                                                                                                                                                                                                                     |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtXml.QDomCDATASection`          | The text is copied to the new node.                                                                                                                                                                                                                                                     |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| :sip:ref:`~PyQt6.QtXml.QDomComment`               | The text is copied to the new node.                                                                                                                                                                                                                                                     |
+---------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
