If *text* is not empty, explicitly assigns *text* as the string to be returned by :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.text`. :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.isTextGenerated` will return false.

If *text* is empty, indicates that :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.text` should be automatically generated from the address elements. :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.isTextGenerated` will return true.
