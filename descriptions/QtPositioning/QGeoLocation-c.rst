The :sip:ref:`~PyQt6.QtPositioning.QGeoLocation` class represents basic information about a location.

A :sip:ref:`~PyQt6.QtPositioning.QGeoLocation` consists of a coordinate and corresponding address, along with an optional bounding shape, which is the recommended region to be displayed when viewing the location.
