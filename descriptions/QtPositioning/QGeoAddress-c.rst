The :sip:ref:`~PyQt6.QtPositioning.QGeoAddress` class represents an address of a :sip:ref:`~PyQt6.QtPositioning.QGeoLocation`.

The address' attributes are normalized to US feature names and can be mapped to the local feature levels (for example State matches "Bundesland" in Germany).

The address contains a :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.text` for displaying purposes and additional properties to access the components of an address:

* :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.country`

* :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.countryCode`

* :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.state`

* :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.city`

* :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.district`

* :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.street`

* :sip:ref:`~PyQt6.QtPositioning.QGeoAddress.postalCode`
