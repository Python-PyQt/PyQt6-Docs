Returns a deep copy of the subset of the pixmap that is specified by the given *rectangle*. For more information on deep copies, see the `Implicit Data Sharing <https://doc.qt.io/qt-6/implicit-sharing.html>`_ documentation.

If the given *rectangle* is empty, the whole image is copied.
