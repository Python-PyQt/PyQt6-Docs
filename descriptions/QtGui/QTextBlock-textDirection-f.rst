Returns the resolved text direction.

If the block has no explicit direction set, it will resolve the direction from the blocks content. Returns either :sip:ref:`~PyQt6.QtCore.Qt.LayoutDirection.LeftToRight` or :sip:ref:`~PyQt6.QtCore.Qt.LayoutDirection.RightToLeft`.
