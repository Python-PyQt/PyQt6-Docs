If *enable* is true sets the font's weight to :sip:ref:`~PyQt6.QtGui.QFont.Weight.Weight`; otherwise sets the weight to :sip:ref:`~PyQt6.QtGui.QFont.Weight.Weight`.

For finer boldness control use :sip:ref:`~PyQt6.QtGui.QFont.setWeight`.

**Note:** If :sip:ref:`~PyQt6.QtGui.QFont.styleName` is set, this value may be ignored, or if supported on the platform, the font artificially embolded.

.. seealso:: :sip:ref:`~PyQt6.QtGui.QFont.bold`, :sip:ref:`~PyQt6.QtGui.QFont.setWeight`.
