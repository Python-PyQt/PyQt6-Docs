Sets whether the item is selectable. If *selectable* is true, the item can be selected by the user; otherwise, the user cannot select the item.

You can control the selection behavior and mode by manipulating their view properties; see :sip:ref:`~PyQt6.QtWidgets.QAbstractItemView.selectionMode` and :sip:ref:`~PyQt6.QtWidgets.QAbstractItemView.selectionBehavior`.
