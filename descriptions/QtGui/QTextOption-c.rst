The :sip:ref:`~PyQt6.QtGui.QTextOption` class provides a description of general rich text properties.

:sip:ref:`~PyQt6.QtGui.QTextOption` is used to encapsulate common rich text properties in a single object. It contains information about text alignment, layout direction, word wrapping, and other standard properties associated with text rendering and layout.
