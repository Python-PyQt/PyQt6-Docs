Sets the :sip:ref:`~PyQt6.QtGui.QFont.style` of the font to :sip:ref:`~PyQt6.QtGui.QFont.Style.StyleItalic` if *enable* is true; otherwise the style is set to :sip:ref:`~PyQt6.QtGui.QFont.Style.StyleNormal`.

**Note:** If :sip:ref:`~PyQt6.QtGui.QFont.styleName` is set, this value may be ignored, or if supported on the platform, the font may be rendered tilted instead of picking a designed italic font-variant.

.. seealso:: :sip:ref:`~PyQt6.QtGui.QFont.italic`, :sip:ref:`~PyQt6.QtGui.QFontInfo`.
