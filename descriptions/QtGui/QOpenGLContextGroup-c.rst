The :sip:ref:`~PyQt6.QtGui.QOpenGLContextGroup` class represents a group of contexts sharing OpenGL resources.

:sip:ref:`~PyQt6.QtGui.QOpenGLContextGroup` is automatically created and managed by `QOpenGLContext <https://doc.qt.io/qt-6/gui-changes-qt6.html#qopenglcontext>`_ instances. Its purpose is to identify all the contexts that are sharing resources.

.. seealso:: :sip:ref:`~PyQt6.QtGui.QOpenGLContext.shareGroup`.
