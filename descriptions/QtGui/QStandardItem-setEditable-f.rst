Sets whether the item is editable. If *editable* is true, the item can be edited by the user; otherwise, the user cannot edit the item.

How the user can edit items in a view is determined by the view's edit triggers; see :sip:ref:`~PyQt6.QtWidgets.QAbstractItemView.editTriggers`.
