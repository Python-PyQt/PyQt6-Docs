Constructs a :sip:ref:`~PyQt6.QtGui.QMovie` object. :sip:ref:`~PyQt6.QtGui.QMovie` will use read image data from *device*, which it assumes is open and readable. If *format* is not empty, :sip:ref:`~PyQt6.QtGui.QMovie` will use the image format *format* for decoding the image data. Otherwise, :sip:ref:`~PyQt6.QtGui.QMovie` will attempt to guess the format.

The *parent* object is passed to :sip:ref:`~PyQt6.QtCore.QObject`'s constructor.
