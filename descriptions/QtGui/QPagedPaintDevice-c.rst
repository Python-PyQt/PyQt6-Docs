The :sip:ref:`~PyQt6.QtGui.QPagedPaintDevice` class represents a paint device that supports multiple pages.

Paged paint devices are used to generate output for printing or for formats like PDF. :sip:ref:`~PyQt6.QtGui.QPdfWriter` and QPrinter inherit from it.
