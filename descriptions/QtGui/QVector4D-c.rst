The :sip:ref:`~PyQt6.QtGui.QVector4D` class represents a vector or vertex in 4D space.

The :sip:ref:`~PyQt6.QtGui.QVector4D` class can also be used to represent vertices in 4D space. We therefore do not need to provide a separate vertex class.

.. seealso:: :sip:ref:`~PyQt6.QtGui.QQuaternion`, :sip:ref:`~PyQt6.QtGui.QVector2D`, :sip:ref:`~PyQt6.QtGui.QVector3D`.
