Returns the tool tip text brush of the current color group. This brush is used by :sip:ref:`~PyQt6.QtWidgets.QToolTip` and :sip:ref:`~PyQt6.QtWidgets.QWhatsThis`.

**Note:** Tool tips use the Inactive color group of :sip:ref:`~PyQt6.QtGui.QPalette`, because tool tips are not active windows.
