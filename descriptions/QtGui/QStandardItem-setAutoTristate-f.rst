Determines that the item is tristate and controlled by :sip:ref:`~PyQt6.QtWidgets.QTreeWidget` if *tristate* is ``true``. This enables automatic management of the state of parent items in :sip:ref:`~PyQt6.QtWidgets.QTreeWidget` (checked if all children are checked, unchecked if all children are unchecked, or partially checked if only some children are checked).
