The :sip:ref:`~PyQt6.QtGui.QPdfWriter` class is a class to generate PDFs that can be used as a paint device.

:sip:ref:`~PyQt6.QtGui.QPdfWriter` generates PDF out of a series of drawing commands using :sip:ref:`~PyQt6.QtGui.QPainter`. The :sip:ref:`~PyQt6.QtGui.QPdfWriter.newPage` method can be used to create several pages.
