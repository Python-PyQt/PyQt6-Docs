Returns the anchor position; this is the same as :sip:ref:`~PyQt6.QtGui.QTextCursor.position` unless there is a selection in which case :sip:ref:`~PyQt6.QtGui.QTextCursor.position` marks one end of the selection and  marks the other end. Just like the cursor position, the anchor position is between characters.
