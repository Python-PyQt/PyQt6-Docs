Static convenience function that returns a :sip:ref:`~PyQt6.QtGui.QColor` constructed from the given CMYK color values: *c* (cyan), *m* (magenta), *y* (yellow), *k* (black), and *a* (alpha-channel, i.e. transparency).

All the values must be in the range 0-255.

.. seealso:: :sip:ref:`~PyQt6.QtGui.QColor.toCmyk`, :sip:ref:`~PyQt6.QtGui.QColor.fromCmykF`, :sip:ref:`~PyQt6.QtGui.QColor.isValid`, The CMYK Color Model.
