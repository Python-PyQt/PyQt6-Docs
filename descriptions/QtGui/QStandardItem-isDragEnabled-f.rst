Returns whether the item is drag enabled. An item that is drag enabled can be dragged by the user.

The default value is true.

Note that item dragging must be enabled in the view for dragging to work; see :sip:ref:`~PyQt6.QtWidgets.QAbstractItemView.dragEnabled`.
