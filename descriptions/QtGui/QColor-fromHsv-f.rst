Static convenience function that returns a :sip:ref:`~PyQt6.QtGui.QColor` constructed from the HSV color values, *h* (hue), *s* (saturation), *v* (value), and *a* (alpha-channel, i.e. transparency).

The value of *s*, *v*, and *a* must all be in the range 0-255; the value of *h* must be in the range 0-359.

.. seealso:: :sip:ref:`~PyQt6.QtGui.QColor.toHsv`, :sip:ref:`~PyQt6.QtGui.QColor.fromHsvF`, :sip:ref:`~PyQt6.QtGui.QColor.isValid`, The HSV Color Model.
