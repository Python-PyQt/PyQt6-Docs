The :sip:ref:`~PyQt6.QtGui.QTextTableCellFormat` class provides formatting information for table cells in a :sip:ref:`~PyQt6.QtGui.QTextDocument`.

The table cell format of a table cell in a document specifies the visual properties of the table cell.

The padding properties of a table cell are controlled by :sip:ref:`~PyQt6.QtGui.QTextTableCellFormat.setLeftPadding`, :sip:ref:`~PyQt6.QtGui.QTextTableCellFormat.setRightPadding`, :sip:ref:`~PyQt6.QtGui.QTextTableCellFormat.setTopPadding`, and :sip:ref:`~PyQt6.QtGui.QTextTableCellFormat.setBottomPadding`. All the paddings can be set at once using :sip:ref:`~PyQt6.QtGui.QTextTableCellFormat.setPadding`.
