Returns whether the item is enabled.

When an item is enabled, the user can interact with it. The possible types of interaction are specified by the other item flags, such as :sip:ref:`~PyQt6.QtGui.QStandardItem.isEditable` and :sip:ref:`~PyQt6.QtGui.QStandardItem.isSelectable`.

The default value is true.

.. seealso:: :sip:ref:`~PyQt6.QtGui.QStandardItem.setEnabled`, :sip:ref:`~PyQt6.QtGui.QStandardItem.flags`.
