The :sip:ref:`~PyQt6.QtGui.QResizeEvent` class contains event parameters for resize events.

Resize events are sent to widgets that have been resized.

The event handler :sip:ref:`~PyQt6.QtWidgets.QWidget.resizeEvent` receives resize events.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QWidget.resize`, :sip:ref:`~PyQt6.QtWidgets.QWidget.setGeometry`.
