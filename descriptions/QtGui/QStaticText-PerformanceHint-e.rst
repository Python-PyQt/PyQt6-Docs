This enum the different performance hints that can be set on the :sip:ref:`~PyQt6.QtGui.QStaticText`. These hints can be used to indicate that the :sip:ref:`~PyQt6.QtGui.QStaticText` should use additional caches, if possible, to improve performance at the expense of memory. In particular, setting the performance hint  on the :sip:ref:`~PyQt6.QtGui.QStaticText` will improve performance when using the OpenGL graphics system or when drawing to a :sip:ref:`~PyQt6.QtOpenGLWidgets.QOpenGLWidget`.
