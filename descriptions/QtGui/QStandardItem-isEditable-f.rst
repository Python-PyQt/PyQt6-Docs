Returns whether the item can be edited by the user.

When an item is editable (and enabled), the user can edit the item by invoking one of the view's edit triggers; see :sip:ref:`~PyQt6.QtWidgets.QAbstractItemView.editTriggers`.

The default value is true.

.. seealso:: :sip:ref:`~PyQt6.QtGui.QStandardItem.setEditable`, :sip:ref:`~PyQt6.QtGui.QStandardItem.flags`.
