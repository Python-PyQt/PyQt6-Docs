Sets the visual cursor movement style to the given *style*. If the :sip:ref:`~PyQt6.QtGui.QTextLayout` is backed by a document, you can ignore this and use the option in :sip:ref:`~PyQt6.QtGui.QTextDocument`, this option is for widgets like QLineEdit or custom widgets without a :sip:ref:`~PyQt6.QtGui.QTextDocument`. Default value is :sip:ref:`~PyQt6.QtCore.Qt.CursorMoveStyle.LogicalMoveStyle`.
