Shows the window as fullscreen.

Equivalent to calling :sip:ref:`~PyQt6.QtGui.QWindow.setWindowStates`\ (\ :sip:ref:`~PyQt6.QtCore.Qt.WindowState.WindowFullScreen`) and then :sip:ref:`~PyQt6.QtGui.QWindow.setVisible`\ (true).

See the :sip:ref:`~PyQt6.QtWidgets.QWidget.showFullScreen` documentation for platform-specific considerations and limitations.
