Returns the smallest rectangle that contains all glyphs in this :sip:ref:`~PyQt6.QtGui.QGlyphRun`. If a bounding rect has been set using :sip:ref:`~PyQt6.QtGui.QGlyphRun.setBoundingRect`, then this will be returned. Otherwise the bounding rect will be calculated based on the font metrics of the glyphs in the glyph run.

.. seealso:: :sip:ref:`~PyQt6.QtGui.QGlyphRun.setBoundingRect`.
