Sets the quality that should be used by exporters when exporting the image. :sip:ref:`~PyQt6.QtGui.QTextDocumentWriter` will export jpg images with the *quality* set here when exporting to ODF files if *quality* is set to a value between 0 and 100. Or it will export png images if *quality* is set to 100 (default) or greater.

.. seealso:: :sip:ref:`~PyQt6.QtGui.QTextImageFormat.quality`.
