The :sip:ref:`~PyQt6.QtGui.QVector2D` class represents a vector or vertex in 2D space.

The :sip:ref:`~PyQt6.QtGui.QVector2D` class can also be used to represent vertices in 2D space. We therefore do not need to provide a separate vertex class.

.. seealso:: :sip:ref:`~PyQt6.QtGui.QVector3D`, :sip:ref:`~PyQt6.QtGui.QVector4D`, :sip:ref:`~PyQt6.QtGui.QQuaternion`.
