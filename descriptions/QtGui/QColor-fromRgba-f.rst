Static convenience function that returns a :sip:ref:`~PyQt6.QtGui.QColor` constructed from the given QRgb value *rgba*.

Unlike the :sip:ref:`~PyQt6.QtGui.QColor.fromRgb` function, the alpha-channel specified by the given QRgb value is included.
