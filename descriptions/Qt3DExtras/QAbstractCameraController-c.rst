The :sip:ref:`~PyQt6.Qt3DExtras.QAbstractCameraController` class provides basic functionality for camera controllers.

:sip:ref:`~PyQt6.Qt3DExtras.QAbstractCameraController` sets up and handles input from keyboard, mouse, and other devices. :sip:ref:`~PyQt6.Qt3DExtras.QAbstractCameraController` is an abstract class and cannot itself be instantiated. It provides a standard interface for camera controllers.
