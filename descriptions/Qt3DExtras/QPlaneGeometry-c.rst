The :sip:ref:`~PyQt6.Qt3DExtras.QPlaneGeometry` class allows creation of a plane in 3D space.

The :sip:ref:`~PyQt6.Qt3DExtras.QPlaneGeometry` class is most commonly used internally by the QPlaneMesh but can also be used in custom :sip:ref:`~PyQt6.Qt3DRender.QGeometryRenderer` subclasses.
