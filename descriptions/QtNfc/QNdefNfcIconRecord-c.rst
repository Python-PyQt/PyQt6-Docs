The :sip:ref:`~PyQt6.QtNfc.QNdefNfcIconRecord` class provides an NFC MIME record to hold an icon.

This class wraps the image data into an NDEF message. It provides an NDEF record of type :sip:ref:`~PyQt6.QtNfc.QNdefRecord.TypeNameFormat.Mime`. The :sip:ref:`~PyQt6.QtNfc.QNdefRecord.payload`\ () contains the raw image data.
