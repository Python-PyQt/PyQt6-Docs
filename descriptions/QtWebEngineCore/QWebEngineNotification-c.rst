The :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineNotification` class encapsulates the data of an HTML5 web notification.

This class contains the information and API for HTML5 desktop and push notifications.

Web engine notifications are passed to the user through the custom handler provided with the :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineProfile.setNotificationPresenter` call.

For more information about how to handle web notification, see the `Notification Example <https://doc.qt.io/qt-6/qtwebengine-webenginewidgets-notifications-example.html>`_.
