A utility type for the :sip:ref:`~PyQt6.QtWebEngineCore.QWebEnginePage.navigationRequested` signal.

Contains information about a navigation request.

To accept or reject a request, call :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineNavigationRequest.accept` or :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineNavigationRequest.reject`.

The default if not handled is to accept the navigation.
