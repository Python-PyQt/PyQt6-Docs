This variable holds A list of server URI templates used for secure DNS-over-HTTPS host resolution..

The ``serverTemplates`` structure member lists `URI templates <https://datatracker.ietf.org/d7oc/html/rfc6570>`_. An example of a URI template is https://dns.google/dns-query{?dns}.
