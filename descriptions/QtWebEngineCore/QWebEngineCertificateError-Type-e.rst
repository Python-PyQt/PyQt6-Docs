This enum describes the type of certificate error encountered.

The values of this enum type match the SSL errors Chromium provides. :sip:ref:`~PyQt6.QtNetwork.QSslError.SslError` values are not used directly, because the Qt error categories cannot be mapped to the Chromium error categories.
