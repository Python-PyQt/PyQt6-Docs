This variable holds The DNS mode used for the host resolution..

Set *secureMode* to :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineGlobalSettings.SecureDnsMode.SecureOnly` to only allow DNS-over-HTTPS host resolution using servers from *serverTemplates*.

Set *secureMode* to :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineGlobalSettings.SecureDnsMode.SecureWithFallback` to enable DNS-over-HTTPS host resolution using servers from *serverTemplates*, with a fallback to the system DNS.

.. seealso:: :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineGlobalSettings.SecureDnsMode`.
