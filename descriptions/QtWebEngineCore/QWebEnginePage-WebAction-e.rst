This enum describes the types of action which can be performed on the web page.

Actions only have an effect when they are applicable.

The availability of actions can be be determined by checking :sip:ref:`~PyQt6.QtGui.QAction.isEnabled` on the action returned by :sip:ref:`~PyQt6.QtWebEngineCore.QWebEnginePage.action`.
