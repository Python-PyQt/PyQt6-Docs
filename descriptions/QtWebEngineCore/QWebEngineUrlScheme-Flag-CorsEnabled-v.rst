Enables cross-origin resource sharing (CORS) for this scheme. This flag is required in order to, for example, use the scheme with the `Fetch API <https://fetch.spec.whatwg.org/>`_, or to deliver CSS fonts to a different origin. The appropriate CORS headers are generated automatically by the :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineUrlRequestJob` class. (Added in Qt 5.14)
