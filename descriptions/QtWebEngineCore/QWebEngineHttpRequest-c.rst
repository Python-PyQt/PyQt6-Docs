The :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineHttpRequest` class holds a request to be sent with `WebEngine <https://doc.qt.io/qt-6/qml-qtwebengine-webengine.html>`_.

:sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineHttpRequest` represents an HTTP request in the `WebEngine <https://doc.qt.io/qt-6/qml-qtwebengine-webengine.html>`_ networking stack. It holds the information necessary to send a request over the network. It contains a URL and some ancillary information that can be used to modify the request. Both :sip:ref:`~PyQt6.QtWebEngineCore.QWebEnginePage.load` and :sip:ref:`~PyQt6.QtWebEngineWidgets.QWebEngineView.load` accept a :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineHttpRequest` as a parameter.
