The authority component of a URL of this type has all of the standard elements: host, port, user name, and password. A URL without a port will use the :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineUrlScheme.defaultPort` (which *must* not be :sip:ref:`~PyQt6.QtWebEngineCore.QWebEngineUrlScheme.SpecialPort.PortUnspecified`).
