Enables a URL scheme to be used by the HTML5 fetch API and ``XMLHttpRequest.send`` with a body. By default only ``http`` and ``https`` can be send to using the Fetch API or with an `XMLHttpRequest <https://doc.qt.io/qt-6/qml-qtqml-xmlhttprequest.html>`_ with a body.
