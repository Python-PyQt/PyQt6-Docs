Returns a :sip:ref:`~PyQt6.QtMultimedia.QMediaTimeRange` describing the currently buffered data.

When streaming media from a remote source, different parts of the media file can be available locally. The returned :sip:ref:`~PyQt6.QtMultimedia.QMediaTimeRange` object describes the time ranges that are buffered and available for immediate playback.
