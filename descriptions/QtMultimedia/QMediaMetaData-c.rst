Provides meta-data for media files.

**Note:** Not all identifiers are supported on all platforms.

+-------------------+
| Common attributes |
+-------------------+

+-------------+--------------------------------------------+-------------------------------------------+
| Value       | Description                                | Type                                      |
+=============+============================================+===========================================+
| Title       | The title of the media.                    | QString                                   |
+-------------+--------------------------------------------+-------------------------------------------+
| Author      | The authors of the media.                  | QStringList                               |
+-------------+--------------------------------------------+-------------------------------------------+
| Comment     | A user comment about the media.            | QString                                   |
+-------------+--------------------------------------------+-------------------------------------------+
| Description | A description of the media.                | QString                                   |
+-------------+--------------------------------------------+-------------------------------------------+
| Genre       | The genre of the media.                    | QStringList                               |
+-------------+--------------------------------------------+-------------------------------------------+
| Date        | The date of the media.                     | :sip:ref:`~PyQt6.QtCore.QDate`.           |
+-------------+--------------------------------------------+-------------------------------------------+
| Language    | The language of media.                     | :sip:ref:`~PyQt6.QtCore.QLocale.Language` |
+-------------+--------------------------------------------+-------------------------------------------+
| Publisher   | The publisher of the media.                | QString                                   |
+-------------+--------------------------------------------+-------------------------------------------+
| Copyright   | The media's copyright notice.              | QString                                   |
+-------------+--------------------------------------------+-------------------------------------------+
| Url         | A Url pointing to the origin of the media. | :sip:ref:`~PyQt6.QtCore.QUrl`             |
+-------------+--------------------------------------------+-------------------------------------------+

+------------------+--------------------------------------------+--------------------------------------------------------+
| Media attributes |                                            |                                                        |
+==================+============================================+========================================================+

|                  | The file format of the media.              | :sip:ref:`~PyQt6.QtMultimedia.QMediaFormat.FileFormat` |
+------------------+--------------------------------------------+--------------------------------------------------------+

+------------------+--------------------------------------------+--------------------------------------------------------+

+------------------+--------------------------------------------------------------+-------------------------+

+------------------+--------------------------------------------------------------+-------------------------+

+------------------+--------------------------------------------------------------+--------------------------------------------------------+
| Video attributes |                                                              |                                                        |
+==================+==============================================================+========================================================+
|                  | The frame rate of the media's video stream.                  | qreal                                                  |
+------------------+--------------------------------------------------------------+--------------------------------------------------------+
|                  | The bit rate of the media's video stream in bits per second. | int                                                    |
+------------------+--------------------------------------------------------------+--------------------------------------------------------+
|                  | The codec of the media's video stream.                       | :sip:ref:`~PyQt6.QtMultimedia.QMediaFormat.VideoCodec` |

|                  | The lead performer in the media.                        | QStringList                    |
