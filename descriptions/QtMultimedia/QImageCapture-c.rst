The :sip:ref:`~PyQt6.QtMultimedia.QImageCapture` class is used for the recording of media content.

The :sip:ref:`~PyQt6.QtMultimedia.QImageCapture` class is a high level images recording class. It's not intended to be used alone but for accessing the media recording functions of other media objects, like :sip:ref:`~PyQt6.QtMultimedia.QCamera`.
