The :sip:ref:`~PyQt6.QtMultimedia.QMediaPlayer` class allows the playing of a media files.

The :sip:ref:`~PyQt6.QtMultimedia.QMediaPlayer` class is a high level media playback class. It can be used to playback audio of video media files. The content to playback is specified as a :sip:ref:`~PyQt6.QtCore.QUrl` object.
