The :sip:ref:`~PyQt6.QtMultimedia.QAudioDecoder` class implements decoding audio.

The :sip:ref:`~PyQt6.QtMultimedia.QAudioDecoder` class is a high level class for decoding audio media files. It is similar to the :sip:ref:`~PyQt6.QtMultimedia.QMediaPlayer` class except that audio is provided back through this API rather than routed directly to audio hardware.
