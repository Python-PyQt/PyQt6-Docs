The :sip:ref:`~PyQt6.QtMultimedia.QCameraFormat` class describes a video format supported by a camera device.

:sip:ref:`~PyQt6.QtMultimedia.QCameraFormat` represents a certain video format supported by a camera device.

The format is a combination of a :sip:ref:`~PyQt6.QtMultimedia.QVideoFrameFormat.PixelFormat`, resolution and a range of frame rates.

:sip:ref:`~PyQt6.QtMultimedia.QCameraFormat` objects can be queried from :sip:ref:`~PyQt6.QtMultimedia.QCameraDevice` to inspect the set of supported video formats.

.. seealso:: :sip:ref:`~PyQt6.QtMultimedia.QCameraDevice`, :sip:ref:`~PyQt6.QtMultimedia.QCamera`.
