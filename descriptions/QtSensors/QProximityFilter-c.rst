The :sip:ref:`~PyQt6.QtSensors.QProximityFilter` class is a convenience wrapper around :sip:ref:`~PyQt6.QtSensors.QSensorFilter`.

The only difference is that the :sip:ref:`~PyQt6.QtSensors.QProximityFilter.filter` method features a pointer to :sip:ref:`~PyQt6.QtSensors.QProximityReading` instead of :sip:ref:`~PyQt6.QtSensors.QSensorReading`.
