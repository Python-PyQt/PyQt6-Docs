The :sip:ref:`~PyQt6.QtSensors.QAmbientLightFilter` class is a convenience wrapper around :sip:ref:`~PyQt6.QtSensors.QSensorFilter`.

The only difference is that the :sip:ref:`~PyQt6.QtSensors.QAmbientLightFilter.filter` method features a pointer to :sip:ref:`~PyQt6.QtSensors.QAmbientLightReading` instead of :sip:ref:`~PyQt6.QtSensors.QSensorReading`.
