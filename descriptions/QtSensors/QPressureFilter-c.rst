The :sip:ref:`~PyQt6.QtSensors.QPressureFilter` class is a convenience wrapper around :sip:ref:`~PyQt6.QtSensors.QSensorFilter`.

The only difference is that the :sip:ref:`~PyQt6.QtSensors.QPressureFilter.filter` method features a pointer to :sip:ref:`~PyQt6.QtSensors.QPressureReading` instead of :sip:ref:`~PyQt6.QtSensors.QSensorReading`.
