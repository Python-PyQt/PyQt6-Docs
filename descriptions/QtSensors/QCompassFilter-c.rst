The :sip:ref:`~PyQt6.QtSensors.QCompassFilter` class is a convenience wrapper around :sip:ref:`~PyQt6.QtSensors.QSensorFilter`.

The only difference is that the :sip:ref:`~PyQt6.QtSensors.QCompassFilter.filter` method features a pointer to :sip:ref:`~PyQt6.QtSensors.QCompassReading` instead of :sip:ref:`~PyQt6.QtSensors.QSensorReading`.
