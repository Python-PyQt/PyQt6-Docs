The :sip:ref:`~PyQt6.QtSensors.QOrientationFilter` class is a convenience wrapper around :sip:ref:`~PyQt6.QtSensors.QSensorFilter`.

The only difference is that the :sip:ref:`~PyQt6.QtSensors.QOrientationFilter.filter` method features a pointer to :sip:ref:`~PyQt6.QtSensors.QOrientationReading` instead of :sip:ref:`~PyQt6.QtSensors.QSensorReading`.
