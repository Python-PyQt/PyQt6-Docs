The :sip:ref:`~PyQt6.QtDataVisualization.QScatterDataProxy` class is the data proxy for 3D scatter graphs.

A scatter data proxy handles adding, inserting, changing, and removing data items.

:sip:ref:`~PyQt6.QtDataVisualization.QScatterDataProxy` takes ownership of all QtDataVisualization::QScatterDataArray and :sip:ref:`~PyQt6.QtDataVisualization.QScatterDataItem` objects passed to it.

.. seealso:: `Qt Data Visualization Data Handling <https://doc.qt.io/qt-6/qtdatavisualization-data-handling.html>`_.
