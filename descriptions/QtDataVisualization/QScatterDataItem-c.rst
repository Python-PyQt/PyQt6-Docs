The :sip:ref:`~PyQt6.QtDataVisualization.QScatterDataItem` class provides a container for resolved data to be added to scatter graphs.

A scatter data item holds the data for a single rendered item in a scatter graph. Scatter data proxies parse data into :sip:ref:`~PyQt6.QtDataVisualization.QScatterDataItem` instances for visualization.

.. seealso:: :sip:ref:`~PyQt6.QtDataVisualization.QScatterDataProxy`, :sip:ref:`~PyQt6.Qt Data Visualization C++ Classes`.
