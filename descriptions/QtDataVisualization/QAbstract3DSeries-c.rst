The :sip:ref:`~PyQt6.QtDataVisualization.QAbstract3DSeries` class is a base class for all data series.

There are inherited classes for each supported series type: :sip:ref:`~PyQt6.QtDataVisualization.QBar3DSeries`, :sip:ref:`~PyQt6.QtDataVisualization.QScatter3DSeries`, and :sip:ref:`~PyQt6.QtDataVisualization.QSurface3DSeries`.

For more information, see `Qt Data Visualization Data Handling <https://doc.qt.io/qt-6/qtdatavisualization-data-handling.html>`_.
