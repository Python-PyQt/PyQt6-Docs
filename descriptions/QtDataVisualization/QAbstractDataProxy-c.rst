The :sip:ref:`~PyQt6.QtDataVisualization.QAbstractDataProxy` class is a base class for all data visualization data proxies.

The following visualization type specific inherited classes are used instead of the base class: :sip:ref:`~PyQt6.QtDataVisualization.QBarDataProxy`, :sip:ref:`~PyQt6.QtDataVisualization.QScatterDataProxy`, and :sip:ref:`~PyQt6.QtDataVisualization.QSurfaceDataProxy`.

For more information, see `Qt Data Visualization Data Handling <https://doc.qt.io/qt-6/qtdatavisualization-data-handling.html>`_.
