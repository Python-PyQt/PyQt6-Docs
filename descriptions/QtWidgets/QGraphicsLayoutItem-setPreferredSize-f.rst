Sets the preferred size to *size*. This property overrides :sip:ref:`~PyQt6.QtWidgets.QGraphicsLayoutItem.sizeHint` for :sip:ref:`~PyQt6.QtCore.Qt.SizeHint.PreferredSize` and provides the default value for :sip:ref:`~PyQt6.QtWidgets.QGraphicsLayoutItem.effectiveSizeHint`. In order to unset the preferred size, use an invalid size.
