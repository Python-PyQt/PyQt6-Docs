Returns this item's flags. The flags describe what configurable features of the item are enabled and not. For example, if the flags include :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.GraphicsItemFlag.ItemIsFocusable`, the item can accept input focus.

By default, no flags are enabled.
