Gets the timeout, in milliseconds, before the gesture triggers.

The recognizer will detect a touch down and if  later the touch is still down, it will trigger the :sip:ref:`~PyQt6.QtWidgets.QTapAndHoldGesture`. The default value is 700 milliseconds.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QTapAndHoldGesture.setTimeout`.
