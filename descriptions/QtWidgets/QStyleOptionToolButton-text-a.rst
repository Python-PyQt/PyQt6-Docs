This variable holds the text of the tool button.

This value is only used if toolButtonStyle is :sip:ref:`~PyQt6.QtCore.Qt.ToolButtonStyle.ToolButtonTextUnderIcon`, :sip:ref:`~PyQt6.QtCore.Qt.ToolButtonStyle.ToolButtonTextBesideIcon`, or :sip:ref:`~PyQt6.QtCore.Qt.ToolButtonStyle.ToolButtonTextOnly`. The default value is an empty string.
