The :sip:ref:`~PyQt6.QtWidgets.QTimeEdit` class provides a widget for editing times based on the :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit` widget.

.. image:: ../../../images/windows-timeedit.png

Many of the properties and functions provided by :sip:ref:`~PyQt6.QtWidgets.QTimeEdit` are implemented in :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit`. These are the relevant properties of this class:

* :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit.time` holds the time displayed by the widget.

* :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit.minimumTime` defines the minimum (earliest) time that can be set by the user.

* :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit.maximumTime` defines the maximum (latest) time that can be set by the user.

* :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit.displayFormat` contains a string that is used to format the time displayed in the widget.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QDateEdit`, :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit`.
