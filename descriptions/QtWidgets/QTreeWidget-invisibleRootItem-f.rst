Returns the tree widget's invisible root item.

The invisible root item provides access to the tree widget's top-level items through the :sip:ref:`~PyQt6.QtWidgets.QTreeWidgetItem` API, making it possible to write functions that can treat top-level items and their children in a uniform way; for example, recursive functions.
