This variable holds the direction of the arrow for the tool button.

This value is only used if features includes :sip:ref:`~PyQt6.QtWidgets.QStyleOptionToolButton.ToolButtonFeatures.Arrow`. The default value is :sip:ref:`~PyQt6.QtCore.Qt.ArrowType.DownArrow`.
