The :sip:ref:`~PyQt6.QtWidgets.QStyleOptionComboBox` class is used to describe the parameter for drawing a combobox.

:sip:ref:`~PyQt6.QtWidgets.QStyleOptionButton` contains all the information that :sip:ref:`~PyQt6.QtWidgets.QStyle` functions need to draw :sip:ref:`~PyQt6.QtWidgets.QComboBox`.

For performance reasons, there are few member functions and the access to the member variables is direct (i.e., using the ``.`` or ``->`` operator). This makes the structures straightforward to use and emphasizes that these are simply parameters used by the style functions.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QStyleOption`, :sip:ref:`~PyQt6.QtWidgets.QStyleOptionComplex`, :sip:ref:`~PyQt6.QtWidgets.QComboBox`.
