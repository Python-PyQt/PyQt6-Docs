Constructs a :sip:ref:`~PyQt6.QtWidgets.QStyleOption` with the specified *version* and *type*.

The version has no special meaning for :sip:ref:`~PyQt6.QtWidgets.QStyleOption`; it can be used by subclasses to distinguish between different version of the same option type.

The state member variable is initialized to :sip:ref:`~PyQt6.QtWidgets.QStyle.State.State_None`.
