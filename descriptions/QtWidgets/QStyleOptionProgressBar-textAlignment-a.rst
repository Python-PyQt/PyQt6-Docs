This variable holds the text alignment for the text in the :sip:ref:`~PyQt6.QtWidgets.QProgressBar`.

This can be used as a guide on where the text should be in the progress bar. The default value is :sip:ref:`~PyQt6.QtCore.Qt.Alignment.AlignLeft`.
