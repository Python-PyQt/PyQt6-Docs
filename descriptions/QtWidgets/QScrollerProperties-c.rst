The :sip:ref:`~PyQt6.QtWidgets.QScrollerProperties` class stores the settings for a :sip:ref:`~PyQt6.QtWidgets.QScroller`.

The :sip:ref:`~PyQt6.QtWidgets.QScrollerProperties` class stores the parameters used by :sip:ref:`~PyQt6.QtWidgets.QScroller`.

The default settings are platform dependent so that Qt emulates the platform behaviour for kinetic scrolling.

As a convention the :sip:ref:`~PyQt6.QtWidgets.QScrollerProperties` are in physical units (meter, seconds) and are converted by :sip:ref:`~PyQt6.QtWidgets.QScroller` using the current DPI.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QScroller`.
