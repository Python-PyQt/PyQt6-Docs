The :sip:ref:`~PyQt6.QtWidgets.QColumnView` class provides a model/view implementation of a column view.

:sip:ref:`~PyQt6.QtWidgets.QColumnView` displays a model in a number of QListViews, one for each hierarchy in the tree. This is sometimes referred to as a cascading list.

The :sip:ref:`~PyQt6.QtWidgets.QColumnView` class is one of the `Model/View Classes <https://doc.qt.io/qt-6/model-view-programming.html#model-view-classes>`_ and is part of Qt's `model/view framework <https://doc.qt.io/qt-6/model-view-programming.html>`_.

:sip:ref:`~PyQt6.QtWidgets.QColumnView` implements the interfaces defined by the :sip:ref:`~PyQt6.QtWidgets.QAbstractItemView` class to allow it to display data provided by models derived from the :sip:ref:`~PyQt6.QtCore.QAbstractItemModel` class.
