This enum type defines the shapes of frame available.

When it does not call :sip:ref:`~PyQt6.QtWidgets.QStyle`, Shape interacts with :sip:ref:`~PyQt6.QtWidgets.QFrame.Shadow`, the :sip:ref:`~PyQt6.QtWidgets.QFrame.lineWidth` and the :sip:ref:`~PyQt6.QtWidgets.QFrame.midLineWidth` to create the total result. See the picture of the frames in the main class documentation.
