Constructs a :sip:ref:`~PyQt6.QtWidgets.QDockWidget` with parent *parent* and window flags *flags*. The dock widget will be placed in the left dock widget area.

The window title is set to *title*. This title is used when the :sip:ref:`~PyQt6.QtWidgets.QDockWidget` is docked and undocked. It is also used in the context menu provided by :sip:ref:`~PyQt6.QtWidgets.QMainWindow`.
