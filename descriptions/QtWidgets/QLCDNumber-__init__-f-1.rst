Constructs an LCD number, sets the number of digits to *numDigits*, the base to decimal, the decimal point mode to 'small' and the frame style to a raised box. The :sip:ref:`~PyQt6.QtWidgets.QLCDNumber.segmentStyle` is set to ``Filled``.

The *parent* argument is passed to the :sip:ref:`~PyQt6.QtWidgets.QFrame` constructor.
