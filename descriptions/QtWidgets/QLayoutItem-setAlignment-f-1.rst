Sets the alignment of this item to *alignment*.

**Note:** Item alignment is only supported by :sip:ref:`~PyQt6.QtWidgets.QLayoutItem` subclasses where it would have a visual effect. Except for :sip:ref:`~PyQt6.QtWidgets.QSpacerItem`, which provides blank space for layouts, all public Qt classes that inherit :sip:ref:`~PyQt6.QtWidgets.QLayoutItem` support item alignment.
