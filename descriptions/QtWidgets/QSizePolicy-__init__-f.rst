Constructs a :sip:ref:`~PyQt6.QtWidgets.QSizePolicy` object with :sip:ref:`~PyQt6.QtWidgets.QSizePolicy.Policy.Fixed` as its horizontal and vertical policies.

The policies can be altered using the :sip:ref:`~PyQt6.QtWidgets.QSizePolicy.setHorizontalPolicy` and :sip:ref:`~PyQt6.QtWidgets.QSizePolicy.setVerticalPolicy` functions. Use the :sip:ref:`~PyQt6.QtWidgets.QSizePolicy.setHeightForWidth` function if the preferred height of the widget is dependent on the width of the widget (for example, a :sip:ref:`~PyQt6.QtWidgets.QLabel` with line wrapping).

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QSizePolicy.setHorizontalStretch`, :sip:ref:`~PyQt6.QtWidgets.QSizePolicy.setVerticalStretch`.
