Sets the positioning mode used by :sip:ref:`~PyQt6.QtWidgets.QGridLayout.addItem`. If *orient* is :sip:ref:`~PyQt6.QtCore.Qt.Orientations.Horizontal`, this layout is expanded to *n* columns, and items will be added columns-first. Otherwise it is expanded to *n* rows and items will be added rows-first.
