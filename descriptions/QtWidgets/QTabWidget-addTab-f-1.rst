This is an overloaded function.

Adds a tab with the given *page*, *icon*, and *label* to the tab widget, and returns the index of the tab in the tab bar. Ownership of *page* is passed on to the :sip:ref:`~PyQt6.QtWidgets.QTabWidget`.

This function is the same as :sip:ref:`~PyQt6.QtWidgets.QTabWidget.addTab`, but with an additional *icon*.
