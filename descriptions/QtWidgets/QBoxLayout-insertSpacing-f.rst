Inserts a non-stretchable space (a :sip:ref:`~PyQt6.QtWidgets.QSpacerItem`) at position *index*, with size *size*. If *index* is negative the space is added at the end.

The box layout has default margin and spacing. This function adds additional space.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QBoxLayout.addSpacing`, :sip:ref:`~PyQt6.QtWidgets.QBoxLayout.insertItem`, :sip:ref:`~PyQt6.QtWidgets.QSpacerItem`.
