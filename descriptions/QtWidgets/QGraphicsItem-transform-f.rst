Returns this item's transformation matrix.

The transformation matrix is combined with the item's :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.rotation`, :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.scale` and :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.transformations` into a combined transformations for the item.

The default transformation matrix is an identity matrix.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.setTransform`, :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.sceneTransform`.
