    :brief: Hover events in the graphics view framework

The :sip:ref:`~PyQt6.QtWidgets.QGraphicsSceneHoverEvent` class provides hover events in the graphics view framework.

When a :sip:ref:`~PyQt6.QtWidgets.QGraphicsView` receives a :sip:ref:`~PyQt6.QtGui.QHoverEvent` event, it translates it into :sip:ref:`~PyQt6.QtWidgets.QGraphicsSceneHoverEvent`. The event is then forwarded to the :sip:ref:`~PyQt6.QtWidgets.QGraphicsScene` associated with the view.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QGraphicsSceneMouseEvent`, :sip:ref:`~PyQt6.QtWidgets.QGraphicsSceneContextMenuEvent`, :sip:ref:`~PyQt6.QtWidgets.QGraphicsSceneWheelEvent`, :sip:ref:`~PyQt6.QtGui.QHoverEvent`.
