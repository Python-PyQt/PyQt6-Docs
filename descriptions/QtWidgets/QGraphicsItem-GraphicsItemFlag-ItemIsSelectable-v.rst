The item supports selection. Enabling this feature will enable :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.setSelected` to toggle selection for the item. It will also let the item be selected automatically as a result of calling :sip:ref:`~PyQt6.QtWidgets.QGraphicsScene.setSelectionArea`, by clicking on an item, or by using rubber band selection in :sip:ref:`~PyQt6.QtWidgets.QGraphicsView`.
