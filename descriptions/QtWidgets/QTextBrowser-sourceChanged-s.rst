This signal is emitted when the source has changed, *src* being the new source.

Source changes happen both programmatically when calling :sip:ref:`~PyQt6.QtWidgets.QTextBrowser.setSource`, :sip:ref:`~PyQt6.QtWidgets.QTextBrowser.forward`, backword() or :sip:ref:`~PyQt6.QtWidgets.QTextBrowser.home` or when the user clicks on links or presses the equivalent key sequences.
