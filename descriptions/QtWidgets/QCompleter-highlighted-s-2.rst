This signal is sent when an item in the :sip:ref:`~PyQt6.QtWidgets.QCompleter.popup` is highlighted by the user. It is also sent if :sip:ref:`~PyQt6.QtWidgets.QCompleter.complete` is called with the :sip:ref:`~PyQt6.QtWidgets.QCompleter.completionMode` set to :sip:ref:`~PyQt6.QtWidgets.QCompleter.CompletionMode.InlineCompletion`. The item's *text* is given.
