Returns the scale factor of the item. The default scale factor is 1.0 (i.e., the item is not scaled).

The scale is combined with the item's :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.rotation`, :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.transform` and :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.transformations` to map the item's coordinate system to the parent item.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.setScale`, :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.rotation`, :ref:`Transformations<qgraphicsitem-transformations>`.
