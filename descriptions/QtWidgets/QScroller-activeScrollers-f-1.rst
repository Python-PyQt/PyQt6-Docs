Returns an application wide list of currently active :sip:ref:`~PyQt6.QtWidgets.QScroller` objects. Active :sip:ref:`~PyQt6.QtWidgets.QScroller` objects are in a :sip:ref:`~PyQt6.QtWidgets.QScroller.state` that is not :sip:ref:`~PyQt6.QtWidgets.QScroller.State.Inactive`. This function is useful when writing your own gesture recognizer.
