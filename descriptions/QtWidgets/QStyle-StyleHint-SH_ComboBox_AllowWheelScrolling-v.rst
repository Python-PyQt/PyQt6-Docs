Determines if the mouse wheel can be used to scroll inside a `QComboBox <https://doc.qt.io/qt-6/widgets-changes-qt6.html#qcombobox>`_. This is on by default in all styles except the Mac style. This enum value has been introduced in Qt 5.10.
