This variable holds the font that is used for the text.

This value is only used if toolButtonStyle is :sip:ref:`~PyQt6.QtCore.Qt.ToolButtonStyle.ToolButtonTextUnderIcon`, :sip:ref:`~PyQt6.QtCore.Qt.ToolButtonStyle.ToolButtonTextBesideIcon`, or :sip:ref:`~PyQt6.QtCore.Qt.ToolButtonStyle.ToolButtonTextOnly`. By default, the application's default font is used.
