Creates and returns a :sip:ref:`~PyQt6.QtWidgets.QStyle` object that matches the given *key*, or returns ``nullptr`` if no matching style is found.

Both built-in styles and styles from style plugins are queried for a matching style.

**Note:** The keys used are case insensitive.
