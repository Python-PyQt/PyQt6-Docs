This enum is used to hold information about the version of the style option, and is defined for each :sip:ref:`~PyQt6.QtWidgets.QStyleHintReturn` subclass.

The version is used by :sip:ref:`~PyQt6.QtWidgets.QStyleHintReturn` subclasses to implement extensions without breaking compatibility. If you use qstyleoption_cast(), you normally do not need to check it.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QStyleHintReturnVariant.StyleOptionType.StyleOptionType`.
