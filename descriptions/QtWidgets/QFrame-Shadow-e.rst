This enum type defines the types of shadow that are used to give a 3D effect to frames.

Shadow interacts with :sip:ref:`~PyQt6.QtWidgets.QFrame.Shape`, the :sip:ref:`~PyQt6.QtWidgets.QFrame.lineWidth` and the :sip:ref:`~PyQt6.QtWidgets.QFrame.midLineWidth`. See the picture of the frames in the main class documentation.
