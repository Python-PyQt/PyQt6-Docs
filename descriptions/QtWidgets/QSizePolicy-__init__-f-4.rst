Constructs a :sip:ref:`~PyQt6.QtWidgets.QSizePolicy` object with the given *horizontal* and *vertical* policies, and the specified control *type*.

Use :sip:ref:`~PyQt6.QtWidgets.QSizePolicy.setHeightForWidth` if the preferred height of the widget is dependent on the width of the widget (for example, a :sip:ref:`~PyQt6.QtWidgets.QLabel` with line wrapping).
