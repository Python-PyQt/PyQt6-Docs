Sets the pixmap for role *which* to *pixmap*.

The pixmaps are used by :sip:ref:`~PyQt6.QtWidgets.QWizard` when displaying a page. Which pixmaps are actually used depend on the :ref:`qwizard-wizard-look-and-feel`.

Pixmaps can also be set for a specific page using :sip:ref:`~PyQt6.QtWidgets.QWizardPage.setPixmap`.
