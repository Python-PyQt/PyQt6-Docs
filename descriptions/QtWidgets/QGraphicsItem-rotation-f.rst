Returns the clockwise rotation, in degrees, around the Z axis. The default value is 0 (i.e., the item is not rotated).

The rotation is combined with the item's :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.scale`, :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.transform` and :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.transformations` to map the item's coordinate system to the parent item.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.setRotation`, :sip:ref:`~PyQt6.QtWidgets.QGraphicsItem.transformOriginPoint`, :ref:`Transformations<qgraphicsitem-transformations>`.
