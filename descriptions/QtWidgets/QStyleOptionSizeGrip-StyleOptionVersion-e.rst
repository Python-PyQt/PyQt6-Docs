This enum is used to hold information about the version of the style option, and is defined for each :sip:ref:`~PyQt6.QtWidgets.QStyleOption` subclass.

The version is used by :sip:ref:`~PyQt6.QtWidgets.QStyleOption` subclasses to implement extensions without breaking compatibility. If you use qstyleoption_cast(), you normally do not need to check it.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QStyleOptionSizeGrip.StyleOptionType.StyleOptionType`.
