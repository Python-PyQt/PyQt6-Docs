The :sip:ref:`~PyQt6.QtWidgets.QStyleOptionToolButton` class is used to describe the parameters for drawing a tool button.

:sip:ref:`~PyQt6.QtWidgets.QStyleOptionToolButton` contains all the information that :sip:ref:`~PyQt6.QtWidgets.QStyle` functions need to draw :sip:ref:`~PyQt6.QtWidgets.QToolButton`.

For performance reasons, there are few member functions and the access to the member variables is direct (i.e., using the ``.`` or ``->`` operator). This makes the structures straightforward to use and emphasizes that these are simply parameters used by the style functions.

For an example demonstrating how style options can be used, see the `Styles <https://doc.qt.io/qt-6/qtwidgets-widgets-styles-example.html>`_ example.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QStyleOption`, :sip:ref:`~PyQt6.QtWidgets.QStyleOptionComplex`, :sip:ref:`~PyQt6.QtWidgets.QStyleOptionButton`.
