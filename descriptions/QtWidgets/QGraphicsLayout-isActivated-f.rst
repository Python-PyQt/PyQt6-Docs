Returns ``true`` if the layout is currently being activated; otherwise, returns ``false``. If the layout is being activated, this means that it is currently in the process of rearranging its items (i.e., the :sip:ref:`~PyQt6.QtWidgets.QGraphicsLayout.activate` function has been called, and has not yet returned).
