Sets the start angle for an ellipse segment to *angle*, which is in 16ths of a degree. This angle is used together with :sip:ref:`~PyQt6.QtWidgets.QGraphicsEllipseItem.spanAngle` for representing an ellipse segment (a pie). By default, the start angle is 0.
