Displays a simple message box about Qt. The message includes the version number of Qt being used by the application.

This is useful for inclusion in the Help menu of an application, as shown in the `Menus <https://doc.qt.io/qt-6/qtwidgets-mainwindows-menus-example.html>`_ example.

This function is a convenience slot for :sip:ref:`~PyQt6.QtWidgets.QMessageBox.aboutQt`.
