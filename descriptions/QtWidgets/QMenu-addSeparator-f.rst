This convenience function creates a new separator action, i.e. an action with :sip:ref:`~PyQt6.QtGui.QAction.isSeparator` returning true, and adds the new action to this menu's list of actions. It returns the newly created action.

:sip:ref:`~PyQt6.QtWidgets.QMenu` takes ownership of the returned :sip:ref:`~PyQt6.QtGui.QAction`.
