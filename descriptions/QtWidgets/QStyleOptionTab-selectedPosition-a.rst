This variable holds the position of the selected tab in relation to this tab.

The default value is :sip:ref:`~PyQt6.QtWidgets.QStyleOptionTab.SelectedPosition.NotAdjacent`, i.e. the tab is not adjacent to a selected tab nor is it the selected tab.
