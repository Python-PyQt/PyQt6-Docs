This variable holds where ellipsis should be added for text that is too long to fit into an item.

The default value is :sip:ref:`~PyQt6.QtCore.Qt.TextElideMode.ElideMiddle`, i.e. the ellipsis appears in the middle of the text.

.. seealso:: :sip:ref:`~PyQt6.QtCore.Qt.TextElideMode`, :sip:ref:`~PyQt6.QtWidgets.QStyle.StyleHint.SH_ItemView_EllipsisLocation`.
