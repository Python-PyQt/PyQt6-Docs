Returns a copy of the :sip:ref:`~PyQt6.QtGui.QTextCursor` that represents the currently visible cursor. Note that changes on the returned cursor do not affect :sip:ref:`~PyQt6.QtWidgets.QPlainTextEdit`'s cursor; use :sip:ref:`~PyQt6.QtWidgets.QPlainTextEdit.setTextCursor` to update the visible cursor.
