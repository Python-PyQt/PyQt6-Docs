This enum contains the different scroll metric types. When not indicated otherwise the :sip:ref:`~PyQt6.QtWidgets.QScrollerProperties.setScrollMetric` function expects a `QVariant <https://doc.qt.io/qt-6/qtcore-changes-qt6.html#qvariant>`_ of type qreal.

See the :sip:ref:`~PyQt6.QtWidgets.QScroller` documentation for further details of the concepts behind the different values.
