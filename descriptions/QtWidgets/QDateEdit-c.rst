The :sip:ref:`~PyQt6.QtWidgets.QDateEdit` class provides a widget for editing dates based on the :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit` widget.

.. image:: ../../../images/windows-dateedit.png

Many of the properties and functions provided by :sip:ref:`~PyQt6.QtWidgets.QDateEdit` are implemented in :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit`. These are the relevant properties of this class:

* :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit.date` holds the date displayed by the widget.

* :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit.minimumDate` defines the minimum (earliest) date that can be set by the user.

* :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit.maximumDate` defines the maximum (latest) date that can be set by the user.

* :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit.displayFormat` contains a string that is used to format the date displayed in the widget.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QTimeEdit`, :sip:ref:`~PyQt6.QtWidgets.QDateTimeEdit`.
