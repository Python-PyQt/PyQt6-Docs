Returns the foreground role.

The foreground role defines the color from the widget's :sip:ref:`~PyQt6.QtWidgets.QWidget.palette` that is used to draw the foreground.

If no explicit foreground role is set, the function returns a role that contrasts with the background role.

.. seealso:: :sip:ref:`~PyQt6.QtWidgets.QWidget.setForegroundRole`, :sip:ref:`~PyQt6.QtWidgets.QWidget.backgroundRole`.
