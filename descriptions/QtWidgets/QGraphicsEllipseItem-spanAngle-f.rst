Returns the span angle of an ellipse segment in 16ths of a degree. This angle is used together with :sip:ref:`~PyQt6.QtWidgets.QGraphicsEllipseItem.startAngle` for representing an ellipse segment (a pie). By default, this function returns 5760 (360 \* 16, a full ellipse).
