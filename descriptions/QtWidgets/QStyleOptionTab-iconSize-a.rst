This variable holds the size for the icons.

The default value is :sip:ref:`~PyQt6.QtCore.QSize`\ (-1, -1), i.e. an invalid size; use :sip:ref:`~PyQt6.QtWidgets.QStyle.pixelMetric` to find the default icon size for tab bars.
