Specifies the text format used to highlight focused anchors in rich text documents displayed for example in :sip:ref:`~PyQt6.QtWidgets.QTextBrowser`. The format has to be a :sip:ref:`~PyQt6.QtGui.QTextCharFormat` returned in the variant of the :sip:ref:`~PyQt6.QtWidgets.QStyleHintReturnVariant` return value. The :sip:ref:`~PyQt6.QtGui.QTextFormat.Property.OutlinePen` property is used for the outline and :sip:ref:`~PyQt6.QtGui.QTextFormat.Property.BackgroundBrush` for the background of the highlighted area.
