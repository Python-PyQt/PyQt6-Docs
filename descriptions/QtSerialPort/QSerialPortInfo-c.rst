Provides information about existing serial ports.

Use the static functions to generate a list of :sip:ref:`~PyQt6.QtSerialPort.QSerialPortInfo` objects. Each :sip:ref:`~PyQt6.QtSerialPort.QSerialPortInfo` object in the list represents a single serial port and can be queried for the port name, system location, description, and manufacturer. The :sip:ref:`~PyQt6.QtSerialPort.QSerialPortInfo` class can also be used as an input parameter for the setPort() method of the :sip:ref:`~PyQt6.QtSerialPort.QSerialPort` class.

.. seealso:: :sip:ref:`~PyQt6.QtSerialPort.QSerialPort`.
