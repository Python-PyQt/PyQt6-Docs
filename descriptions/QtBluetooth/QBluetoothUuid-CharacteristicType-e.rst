This enum is a convienience type for Bluetooth low energy service characteristics class UUIDs. Values of this type will be implicitly converted into a :sip:ref:`~PyQt6.QtBluetooth.QBluetoothUuid` when necessary. The detailed type descriptions can be found on `bluetooth.org <https://developer.bluetooth.org/gatt/characteristics/Pages/CharacteristicsHome.aspx>`_.
