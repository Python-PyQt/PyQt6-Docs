Descriptors are attributes that describe Bluetooth Low Energy characteristic values.

This enum is a convienience type for descriptor class UUIDs. Values of this type will be implicitly converted into a :sip:ref:`~PyQt6.QtBluetooth.QBluetoothUuid` when necessary. The detailed type specifications can be found on `bluetooth.org <https://developer.bluetooth.org/gatt/descriptors/Pages/DescriptorsHomePage.aspx>`_.
