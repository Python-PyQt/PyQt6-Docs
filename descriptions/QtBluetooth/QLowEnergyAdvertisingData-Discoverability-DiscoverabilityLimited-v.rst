The advertising device wishes to be discoverable with a high priority. Note that this mode is not compatible with using a white list. The value of :sip:ref:`~PyQt6.QtBluetooth.QLowEnergyAdvertisingParameters.filterPolicy` is always assumed to be :sip:ref:`~PyQt6.QtBluetooth.QLowEnergyAdvertisingParameters.FilterPolicy.IgnoreWhiteList` when limited discoverability is used.
