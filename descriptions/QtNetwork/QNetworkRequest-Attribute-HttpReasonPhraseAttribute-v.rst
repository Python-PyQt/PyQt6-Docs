Replies only, type: :sip:ref:`~PyQt6.QtCore.QMetaType.Type.QByteArray` (no default) Indicates the HTTP reason phrase as received from the HTTP server (like "Ok", "Found", "Not Found", "Access Denied", etc.) This is the human-readable representation of the status code (see above). If the connection was not HTTP-based, this attribute will not be present. *Note:* The reason phrase is not used when using HTTP/2.
