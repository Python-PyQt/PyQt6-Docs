Requests only, type: :sip:ref:`~PyQt6.QtCore.QMetaType.Type.Bool` (default: false) Indicates whether the :sip:ref:`~PyQt6.QtNetwork.QNetworkAccessManager` code is allowed to buffer the upload data, e.g. when doing a HTTP POST. When using this flag with sequential upload data, the :sip:ref:`~PyQt6.QtNetwork.QNetworkRequest.KnownHeaders.ContentLengthHeader` header must be set.
