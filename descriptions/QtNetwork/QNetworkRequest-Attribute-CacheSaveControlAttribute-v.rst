Requests only, type: :sip:ref:`~PyQt6.QtCore.QMetaType.Type.Bool` (default: true) Controls if the data obtained should be saved to cache for future uses. If the value is false, the data obtained will not be automatically cached. If true, data may be cached, provided it is cacheable (what is cacheable depends on the protocol being used).
