The :sip:ref:`~PyQt6.QtNetwork.QNetworkRequest` class holds a request to be sent with :sip:ref:`~PyQt6.QtNetwork.QNetworkAccessManager`.

:sip:ref:`~PyQt6.QtNetwork.QNetworkRequest` is part of the Network Access API and is the class holding the information necessary to send a request over the network. It contains a URL and some ancillary information that can be used to modify the request.
