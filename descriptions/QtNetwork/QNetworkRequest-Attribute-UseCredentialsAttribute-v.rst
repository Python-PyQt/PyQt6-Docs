Requests only, type: :sip:ref:`~PyQt6.QtCore.QMetaType.Type.Bool` (default: false) Indicates if the underlying `XMLHttpRequest <https://doc.qt.io/qt-6/qml-qtqml-xmlhttprequest.html>`_ cross-site Access-Control requests should be made using credentials. Has no effect on same-origin requests. This only affects the WebAssembly platform. (This value was introduced in 6.5.)
