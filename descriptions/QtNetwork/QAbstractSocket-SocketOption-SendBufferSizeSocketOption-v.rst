Sets the socket send buffer size in bytes at the OS level. This maps to the SO_SNDBUF socket option. This option does not affect the :sip:ref:`~PyQt6.QtCore.QIODevice` or :sip:ref:`~PyQt6.QtNetwork.QAbstractSocket` buffers. This enum value has been introduced in Qt 5.3.
