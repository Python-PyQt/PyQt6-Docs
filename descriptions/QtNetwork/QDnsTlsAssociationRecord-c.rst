The :sip:ref:`~PyQt6.QtNetwork.QDnsTlsAssociationRecord` class stores information about a DNS TLSA record.

When performing a text lookup, zero or more records will be returned. Each record is represented by a :sip:ref:`~PyQt6.QtNetwork.QDnsTlsAssociationRecord` instance.

The meaning of the fields is defined in RFC 6698.

.. seealso:: :sip:ref:`~PyQt6.QtNetwork.QDnsLookup`.
