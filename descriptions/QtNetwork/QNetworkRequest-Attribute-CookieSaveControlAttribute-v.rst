Requests only, type: :sip:ref:`~PyQt6.QtCore.QMetaType.Type.Int` (default: :sip:ref:`~PyQt6.QtNetwork.QNetworkRequest.LoadControl.Automatic`) Indicates whether to save 'Cookie' headers received from the server in reply to the request. This attribute is set to false by Qt WebKit when creating a cross-origin `XMLHttpRequest <https://doc.qt.io/qt-6/qml-qtqml-xmlhttprequest.html>`_ where withCredentials has not been set explicitly to true by the Javascript that created the request. See `here <http://www.w3.org/TR/XMLHttpRequest2/#credentials-flag>`_ for more information. (This value was introduced in 4.7.)
