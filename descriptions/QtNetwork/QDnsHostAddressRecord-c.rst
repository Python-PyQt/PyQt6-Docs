The :sip:ref:`~PyQt6.QtNetwork.QDnsHostAddressRecord` class stores information about a host address record.

When performing an address lookup, zero or more records will be returned. Each record is represented by a :sip:ref:`~PyQt6.QtNetwork.QDnsHostAddressRecord` instance.
