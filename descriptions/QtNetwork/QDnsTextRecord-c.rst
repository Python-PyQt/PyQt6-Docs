The :sip:ref:`~PyQt6.QtNetwork.QDnsTextRecord` class stores information about a DNS TXT record.

When performing a text lookup, zero or more records will be returned. Each record is represented by a :sip:ref:`~PyQt6.QtNetwork.QDnsTextRecord` instance.

The meaning of the fields is defined in `RFC 1035 <http://www.rfc-editor.org/rfc/rfc1035.txt>`_.

.. seealso:: :sip:ref:`~PyQt6.QtNetwork.QDnsLookup`.
