The :sip:ref:`~PyQt6.QtNetwork.QDnsDomainNameRecord` class stores information about a domain name record.

When performing a name server lookup, zero or more records will be returned. Each record is represented by a :sip:ref:`~PyQt6.QtNetwork.QDnsDomainNameRecord` instance.
