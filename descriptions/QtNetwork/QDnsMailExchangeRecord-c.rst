The :sip:ref:`~PyQt6.QtNetwork.QDnsMailExchangeRecord` class stores information about a DNS MX record.

When performing a lookup on a service, zero or more records will be returned. Each record is represented by a :sip:ref:`~PyQt6.QtNetwork.QDnsMailExchangeRecord` instance.

The meaning of the fields is defined in `RFC 1035 <http://www.rfc-editor.org/rfc/rfc1035.txt>`_.

.. seealso:: :sip:ref:`~PyQt6.QtNetwork.QDnsLookup`.
