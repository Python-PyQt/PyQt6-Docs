The :sip:ref:`~PyQt6.QtNetwork.QAbstractNetworkCache` class provides the interface for cache implementations.

:sip:ref:`~PyQt6.QtNetwork.QAbstractNetworkCache` is the base class for every standard cache that is used by :sip:ref:`~PyQt6.QtNetwork.QNetworkAccessManager`. :sip:ref:`~PyQt6.QtNetwork.QAbstractNetworkCache` is an abstract class and cannot be instantiated.
