Special type. Additional information can be passed in QVariants with types ranging from User to UserMax. The default implementation of Network Access will ignore any request attributes in this range and it will not produce any attributes in this range in replies. The range is reserved for extensions of :sip:ref:`~PyQt6.QtNetwork.QNetworkAccessManager`.
