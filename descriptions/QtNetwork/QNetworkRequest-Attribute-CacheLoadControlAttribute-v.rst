Requests only, type: :sip:ref:`~PyQt6.QtCore.QMetaType.Type.Int` (default: :sip:ref:`~PyQt6.QtNetwork.QNetworkRequest.CacheLoadControl.PreferNetwork`) Controls how the cache should be accessed. The possible values are those of :sip:ref:`~PyQt6.QtNetwork.QNetworkRequest.CacheLoadControl`. Note that the default :sip:ref:`~PyQt6.QtNetwork.QNetworkAccessManager` implementation does not support caching. However, this attribute may be used by certain backends to modify their requests (for example, for caching proxies).
