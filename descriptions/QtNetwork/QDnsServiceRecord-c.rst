The :sip:ref:`~PyQt6.QtNetwork.QDnsServiceRecord` class stores information about a DNS SRV record.

When performing a lookup on a service, zero or more records will be returned. Each record is represented by a :sip:ref:`~PyQt6.QtNetwork.QDnsServiceRecord` instance.

The meaning of the fields is defined in `RFC 2782 <http://www.rfc-editor.org/rfc/rfc2782.txt>`_.

.. seealso:: :sip:ref:`~PyQt6.QtNetwork.QDnsLookup`.
