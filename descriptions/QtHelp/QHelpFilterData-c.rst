The :sip:ref:`~PyQt6.QtHelp.QHelpFilterData` class provides details for the filters used by :sip:ref:`~PyQt6.QtHelp.QHelpFilterEngine`.

By using :sip:ref:`~PyQt6.QtHelp.QHelpFilterData.setComponents` you may constrain the search results to documents that belong only to components specified on the given list. By using :sip:ref:`~PyQt6.QtHelp.QHelpFilterData.setVersions` you may constrain the search results to documents that belong only to versions specified on the given list.

.. seealso:: :sip:ref:`~PyQt6.QtHelp.QHelpFilterEngine`.
