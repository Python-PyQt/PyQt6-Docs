The :sip:ref:`~PyQt6.QtHelp.QHelpSearchResult` class provides the data associated with the search result.

The :sip:ref:`~PyQt6.QtHelp.QHelpSearchResult` object is a data object that describes a single search result. The vector of search result objects is returned by :sip:ref:`~PyQt6.QtHelp.QHelpSearchEngine.searchResults`. The description of the search result contains the document title and URL that the search input matched. It also contains the snippet from the document content containing the best match of the search input.

.. seealso:: :sip:ref:`~PyQt6.QtHelp.QHelpSearchEngine`.
